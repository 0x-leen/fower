id: appearance
title: Appearance
---

| Atomic Props   | CSS Properties   |
| -------------- | ---------------- |
| appearanceNone | appearance: none |

## Usage

```jsx live
<Box spaceX3>
  <Box as="select" appearanceNone>
    <option>red</option>
    <option>green</option>
  </Box>

  <select>
    <option>red</option>
    <option>green</option>
  </select>
</Box>
```
