title: Padding
---

Control the padding of an element using the `p{t|r|b|l|x|y}-{size}` Atomic Props.

### Padding Props

`padding` is a style attribute used frequently, so we use abbreviations.

- `p` padding
- `pt` padding-top
- `pr` padding-right
- `pb` padding-bottom
- `pl` padding-left
- `px` padding-left and padding-right
- `py` padding-top and padding-bottom

### Padding of all sides

Control the padding on all sides of an element using the `p-{size}` Atomic Props.

```jsx live
<Text p-10 bgRed300>
  Hello
</Text>
```

### Padding of one side

Control the padding on one side of an element using the `p{t|r|b|l}-{size}` Atomic Props.

```jsx live
<View toCenterX>
  <View debug pt-10>
    Hello
  </View>
  <View debug pr-10>
    Hello
  </View>
  <View debug pb-10>
    Hello
  </View>
  <View debug pl-10>
    Hello
  </View>
</View>
```

### Horizontal padding

Control the horizontal padding of an element using the `px-{size}` Atomic Props.

```jsx live
<Text px-10 bgRed300>
  Hello
</Text>
```

### vertical padding

Control the vertical padding of an element using the `py-{size}` Atomic Props.

```jsx live
<Text py-10 bgRed300>
  Hello
</Text>
```
