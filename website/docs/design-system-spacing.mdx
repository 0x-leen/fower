title: 间距
---

一个应用，元素的间距应该是有规律的，不应该是随意杂乱的，一致的间距可以提高 UI 的设计质量，让用户感受到产品的专业性。
Styli 默认的布局的间距已 4px 为基本单位，所有间距为 4px 的倍数，建议所有的组件和文字排版都使用这些有规律的间距。

## 配置

Styli 默认以 4px 作为间距单元，当然你可以配置自己的间距，修改 `theme.spacing` 即可修改默认的间距。

```jsx
const myPreset = {
  theme: {
    spacing: [0, 4, 8, 12, 16, 20, 24, 28, 32, 36],
  },
}
```

## 用法

间距怎么用呢？使用用法是 `p{spacing}`、`m{spacing}`、`w{spacing}`、`h{spacing}`、`s{spacing}`、`c{spacing}`。

下面我们看一个示例:

```jsx live
<View centerX>
  <View s-100 bgRed40 mr2></View>
  <View s-100 bgYellow40 mr2></View>
  <View s-100 bgGreen40></View>
</View>
```

注意上面例子中的 `mr2`，它就是了设计系统中预设的间距，它相当于 `margin-right: 8px`。

间距可以使用于 padding、margin、width、height、square、circle 五个属性，比如你可以使用这些属性， `p2`、`mx2`、`w4`、`h4`、`s8`、`c8`，并且你可以享受到完美的类型提示。

## 对比精确值

某些情况，设计系统的间距并不能满足你 UI 需求，这是你可使用一种更灵活的方式：精确值，通过中杠 `-` 可以设置任何数值，比如下面的 `s-100`、`mr-20`，会生产样式 `width: 100px; height: 100px`、`margin-right: 20px`：

```jsx live
<View centerX>
  <View s-100 bgRed40 mr-20></View>
  <View s-100 bgYellow40 mr-20></View>
  <View s-100 bgGreen40></View>
</View>
```

大部分情况，建议在项目中使用有规律的间距，这会让你的 UI 更有设计感，更有韵律.
