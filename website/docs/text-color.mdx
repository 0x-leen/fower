title: Text Color

| Atomic Props  | CSS Properties        |
| ------------- | --------------------- |
| {colorName}   | color: {named color}; |
| color={value} | color: {value};       |

## 使用设计系统的颜色

```jsx live
<View f-20>
  <Text gray600>王勃</Text>
  <Text colorPrimary> 李白</Text>
  <Text green400> 杜甫</Text>
</View>
```

所有的预设颜色：[Color palette](/docs/design-system-color)

## 使用 `color` props

```jsx live
<View f-20>
  <Text color="orange400">王勃</Text>
  <Text color="deeppink"> 李白</Text>
  <Text color="#4fd1c5"> 杜甫</Text>
</View>
```

## 颜色透明度

transparentize color 变更透明

```jsx live
<View row toEvenly>
  <View orange500>orange500</View>
  <View orange500-T40>orange500-T40</View>
  <View color="red500">red500</View>
  <View color="red500-T40">red500-T40</View>
  <View color="#4fd1c5">#4fd1c5</View>
  <View color="#4fd1c5-T40">#4fd1c5-T40</View>
</View>
```

## 颜色不透明度

opacify color 变不透明

```jsx live
<View row toEvenly>
  <View color="rgba(0,0,0,0.2)">rgba(0,0,0,0.2)</View>
  <View color="rgba(0,0,0,0.2)-O40">rgba(0,0,0,0.2)-O40</View>
  <View color="rgba(0,0,0,0.2)-O60">rgba(0,0,0,0.2)-O60</View>
  <View color="rgba(0,0,0,0.2)-O80">rgba(0,0,0,0.2)-O80</View>
</View>
```

## 颜色变亮

lighten color 变亮

```jsx live
<View row toEvenly>
  <View orange500>orange500</View>
  <View orange500-L40>orange500-L40</View>
  <View color="red500">red500</View>
  <View color="red500-L40">red500-L40</View>
  <View color="#4fd1c5">#4fd1c5</View>
  <View color="#4fd1c5-L40">#4fd1c5-L40</View>
</View>
```

## 颜色变暗

darken color 变暗

```jsx live
<View row toEvenly>
  <View orange500>orange500</View>
  <View orange500-D30>orange500-D30</View>
  <View color="red500">red500</View>
  <View color="red500-D30">red500-D30</View>
  <View color="#4fd1c5">#4fd1c5</View>
  <View color="#4fd1c5-D30">#4fd1c5-D30</View>
</View>
```

## Hover

```jsx live
<View f-20 row toEvenly>
  <Text green400 red400--hover>
    Hello
  </Text>
  <Text color="#4fd1c5" color--hover="purple">
    Hello
  </Text>
</View>
```

hover with transparentize, lighten, darken:

```jsx live
<View f-20 row toEvenly>
  <Text green400 green400-T60--hover>
    Hello
  </Text>
  <Text green400 green400-L30--hover>
    Hello
  </Text>
  <Text green400 green400-D30--hover>
    Hello
  </Text>
</View>
```

## Focus

```jsx live
<Box as="input" defaultValue="Focus me" outlineNone border p2 green400 red400--focus />
```
