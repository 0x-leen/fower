## Use color in preset

```jsx live
<View f-20 toEvenly>
  <Text gray600>gray600</Text>
  <Text colorPrimary>colorPrimary</Text>
  <Text green400>green400</Text>
</View>
```

call preset colorï¼š[Color palette](/docs/design-system-color#color-palette)

## Use `color` props

```jsx live
<View f-20 toEvenly>
  <Text color="orange400">orange400</Text>
  <Text color="deeppink">deeppink</Text>
  <Text color="#4fd1c5">#4fd1c5</Text>
</View>
```

## Color opacity

transparentize color:

```jsx live
<View row toEvenly>
  <View orange500>orange500</View>
  <View orange500-T40>orange500-T40</View>
  <View color="red500">red500</View>
  <View color="red500-T40">red500-T40</View>
  <View color="#4fd1c5">#4fd1c5</View>
  <View color="#4fd1c5-T40">#4fd1c5-T40</View>
</View>
```

## opacify color

```jsx live
<View row toEvenly>
  <View color="rgba(0,0,0,0.2)">rgba(0,0,0,0.2)</View>
  <View color="rgba(0,0,0,0.2)-O40">rgba(0,0,0,0.2)-O40</View>
  <View color="rgba(0,0,0,0.2)-O60">rgba(0,0,0,0.2)-O60</View>
  <View color="rgba(0,0,0,0.2)-O80">rgba(0,0,0,0.2)-O80</View>
</View>
```

## lighten color

```jsx live
<View row toEvenly>
  <View orange500>orange500</View>
  <View orange500-L40>orange500-L40</View>
  <View color="red500">red500</View>
  <View color="red500-L40">red500-L40</View>
  <View color="#4fd1c5">#4fd1c5</View>
  <View color="#4fd1c5-L40">#4fd1c5-L40</View>
</View>
```

## darken color

```jsx live
<View row toEvenly>
  <View orange500>orange500</View>

  <View color="red500">red500</View>
  <View color="red500-D30">red500-D30</View>
```

## Hover

```jsx live
<View f-20 row toEvenly>
  <Text green400 red400--hover>
    Hello
  </Text>
  <Text color="#4fd1c5" color--hover="purple">
    Hello
  </Text>
</View>
```

hover with transparentize, lighten, darken:

```jsx live
<View f-20 row toEvenly>
  <Text green400 green400-T60--hover>
    Hello
  </Text>
  <Text green400 green400-L30--hover>
    Hello
  </Text>
  <Text green400 green400-D30--hover>
    Hello
  </Text>
</View>
```
