Atomic Props to make children elements align to left.

import { Layout3 } from '@site/src/components/Layout'

| Atomic Props | Effect                               |
| ------------ | ------------------------------------ |
| toLeft       | Make children elements align to left |

## Usage

Use `toTop` to make children elements align to left.

```jsx live
<Box toLeft bgGray100 square-200>
  <Box square-60 bgOrange400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```
