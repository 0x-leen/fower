Atomic Props to make children elements align to left.

import { LayoutEngine3 } from '@site/src/components/LayoutEngine'

| Atomic Props | Effect                               |
| ------------ | ------------------------------------ |
| toLeft       | Make children elements align to left |

## Usage

Use `toTop` to make children elements align to left.

```jsx live
<Box toLeft bgGray100 s-200>
  <Box s-60 bgAmber400 rounded-8></Box>
  <Box s-80 bgBlue400 rounded-8></Box>
</Box>
```
