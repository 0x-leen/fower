使用 c-{size} 格式的 Props 快速把元素变为圆形。

### 用法

```jsx
<View c-50>Hello</View>
<View c-50rem>Hello</View>
<View c={50}>Hello</View>
<View c="50px">Hello</View>
```

### Demo

<Circle />

<TabType
  items={[
    {
      type: 'react',
      value: (
        <JSXHighLight
          code={`
<View c-100></View>
<View c-10rem></View>
<View c="10px"></View>
<View c={10}></View>
`}
        />
      ),
    },
  ]}
/>
