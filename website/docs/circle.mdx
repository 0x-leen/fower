| Atomic Props                    | CSS Properties                                                    |
| ------------------------------- | ----------------------------------------------------------------- |
| circle-{value} / circle={value} | width: {value}; <br/>height: {value};<br/> border-radius: {value} |

### Usage

```jsx live
<View toLeft>
  <View circle-50 bgYellow200 toCenter>
    Hello
  </View>
  <View circle-5rem bgGreen200 toCenter>
    Hello
  </View>
  <View circle={100} bgBlue200 toCenter>
    Hello
  </View>
  <View circle="150px" bgPurple200 toCenter>
    Hello
  </View>
</View>
```
