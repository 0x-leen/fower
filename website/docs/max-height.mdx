id: max-height
title: Max Height
---

使用 maxH-{value} 格式的 Props 可以控制元素的最大高度。

| Atomic Props                | CSS Properties       |
| --------------------------- | -------------------- |
| maxH-{value} / maxH={value} | max-height: {value}; |

## Usage

```jsx live
<View bgGray200 white s-200>
  <View maxH-100 h-200 bgRed400 toCenter>
    maxH-100
  </View>
</View>
```
