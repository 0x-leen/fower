Atomic Props to make children elements align to right.

import { LayoutEngine4 } from '@site/src/components/LayoutEngine'

| Atomic Props | Effect                                |
| ------------ | ------------------------------------- |
| toRight      | Make children elements align to right |

## Usage

Use `toTop` to make children elements align to right.

```jsx live
<Box toRight bgGray100 s-200>
  <Box s-60 bgAmber400 rounded-8></Box>
  <Box s-80 bgBlue400 rounded-8></Box>
</Box>
```
