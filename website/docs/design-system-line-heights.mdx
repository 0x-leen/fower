title: Line heights
---

Fower provides a set of preset line heights values.

```jsx
const config = {
  theme: {
    lineHeights: {
      none: 1,
      tight: 1.25,
      snug: 1.375,
      normal: 1.5,
      relaxed: 1.625,
      loose: 2,
    },
  },
}
```
