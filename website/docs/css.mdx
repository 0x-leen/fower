title: css
---

可以使用 css() 函数创建样式，然后把样式使用在非 Styli 组件上。

使用 modifiers:

```jsx
import React from 'react'
import { css } from '@styli/core'

const cardStyle = css('p-20 s-400 bgRed')

export default () => {
  return <div style={cardStyle}></div>
}
```

还可以使用 CSS 对象:

```jsx
import React from 'react'
import { css } from '@styli/core'

const cardStyle = css('p-20 s-400 bgRed', {
  background: '#fff',
  fontSize: 14,
})

export default () => {
  return <div style={cardStyle}></div>
}
```
