---
id: gradient
title: Gradient
---

Atomic props for control the font style of an element.

| Atomic Props      | CSS Properties                                                |
| ----------------- | ------------------------------------------------------------- |
| gradientX={value} | background-image: linear-gradient(to right, color1, color2);  |
| gradientY={value} | background-image: linear-gradient(to bottom, color1, color2); |

## gradientX

```jsx live
<Box spaceX3 toLeft>
  <Box square20 rounded gradientX={['red400', 'yellow400']} />
  <Box h20 w-200 rounded gradientX={['purple500', 'pink500', 'red500']} />
</Box>
```

hover:

```jsx live
<Box
  cursorPointer
  h20
  w-200
  rounded
  gradientX={['green400', 'blue500']}
  gradientX--hover={['pink500', 'yellow500']}
/>
```

## gradientY

```jsx live
<Box spaceX3 toLeft>
  <Box w20 h-160 rounded gradientY={['red400', 'yellow400']} />
  <Box w20 h-160 rounded gradientY={['purple500', 'red500', 'yellow500']} />
</Box>
```

active:

```jsx live
<Box
  cursorPointer
  square20
  rounded
  gradientY={['green400', 'blue500']}
  gradientY--active={['pink500', 'yellow500']}
/>
```
