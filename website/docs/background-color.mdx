## 使用设计系统的颜色

```jsx live
<View row center>
  <View s-100 bgGreen30></View>
  <View s-100 bgBlue30></View>
  <View s-100 bgPurple30></View>
</View>
```

## 自定义颜色

```jsx live
<View row center>
  <View s-100 bg="#ed8936"></View>
  <View s-100 bg="rgba(44,200, 400)"></View>
  <View s-100 bg="indigo"></View>
</View>
```

## 速改变颜色透明度

```jsx live
<View row center>
  <View s-100 bg="#ed8936"></View>
  <View s-100 bg="#ed8936.8"></View>
  <View s-100 bg="#ed8936.6"></View>
  <View s-100 bg="#ed8936.4"></View>
  <View s-100 bg="indigo"></View>
  <View s-100 bg="indigo.2"></View>
</View>
```
