import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic props for controlling the direction of flex items.

| Atomic Props          | CSS Properties          |
| --------------------- | ----------------------- |
| row                   | flex-direction: row;    |
| column                | flex-direction: column; |
| flexDirection={value} | flex-direction: {value} |

## row

Use `row` props to position flex items horizontally.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box white spaceX3 row>
  <Box bgOrange400 p10>
    A
  </Box>
  <Box bgOrange400 p10>
    B
  </Box>
  <Box bgOrange400 p10>
    C
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div white spaceX3 row>
  <div bgOrange400 p10>A</div>
  <div bgOrange400 p10>B</div>
  <div bgOrange400 p10>C</div>
</div>
```

</TabItem>
</Tabs>

## flexDirection

Use `flexDirection={value}` to set items direction.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box white flex flexDirection="row-reverse">
  <Box bgOrange400 p6 m3>
    A
  </Box>
  <Box bgOrange400 p6 m3>
    B
  </Box>
  <Box bgOrange400 p6 m3>
    C
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div white flex flexDirection="row-reverse">
  <div bgOrange400 p6 m3>A</div>
  <div bgOrange400 p6 m3>B</div>
  <div bgOrange400 p6 m3>C</div>
</div>
```

</TabItem>
</Tabs>
