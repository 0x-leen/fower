## 原子属性

原子类是简单的单一用途的样式单位。与内联样式很像，原子样式仅应用单个样式声明。与内联样式不同，原子样式具有较低的特异性，使其易于覆盖，并且可以通过使用伪类，媒体查询等进行修改。

## Type Safe

我们团队是 TypeScript 的深度使用者，夸张是到了离开 Type Safe 就不会写代码的地步。Styli 是完全类型安全的，特别是配合 JSX，开发体验会非常棒，类型提供很好的智能提示和文档，且不需借助特定的编辑器插件实现智能提示。

![type-safe](/img/type-safe.png)

Styli 的类型安全能带来哪些好处呢？

- 文档作用

  非常多的 css 框架，例如 tailwincss、bootstrap，在熟练使用它们之前，你需要大量翻阅官方文档，而是用 Styli，类型定义就是文档，上面有对每一个 utility 介绍，官方文档更多只是辅助。例如你想给某个容器设置背景颜色，你只要敲下 bg 编辑就会提示所有可用的背景颜色。

- 只能提示

  Styli 提供的类型定义，让编辑器的智能提示更精确，而不需要借助第三方插件

- 易于上手

  类型的智能提示让你更容易探索 Styli 里的所有 utility，大大降低学习成本。
