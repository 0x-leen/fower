## 使用设计系统的颜色

```jsx live
<Box row toCenter>
  <Box square-100 bgGreen300></Box>
  <Box square-100 bgBlue300></Box>
  <Box square-100 bgPurple300></Box>
</Box>
```

## 自定义颜色

```jsx live
<Box row toCenter>
  <Box square-100 bg="#ed8936"></Box>
  <Box square-100 bg="rgba(44,200, 400)"></Box>
  <Box square-100 bg="indigo"></Box>
</Box>
```

## 速改变颜色透明度

```jsx live
<Box row toCenter>
  <Box square-100 bg="#ed8936"></Box>
  <Box square-100 bg="#ed8936.8"></Box>
  <Box square-100 bg="#ed8936.6"></Box>
  <Box square-100 bg="#ed8936.4"></Box>
  <Box square-100 bg="indigo"></Box>
  <Box square-100 bg="indigo.2"></Box>
</Box>
```
