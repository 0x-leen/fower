## Installation

```bash npm2yarn
npm install @fower/taro
```

## Usage

```tsx
import { View, Image, Text } from '@fower/taro'
```

```jsx live
<View toCenterY p-10 w-260 bgWhite rounded-10 shadow>
  <Image circle-48 src="/img/jobs.jpg" />
  <View ml-10>
    <View textXL fontBold>
      Steve Jobs
    </View>
    <Text gray800>Co-founder of Apple Inc.</Text>
  </View>
</View>
```

More documentation is available at [use-with-taro](/docs/use-with-taro).
