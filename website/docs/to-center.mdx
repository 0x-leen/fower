title: To Center
---

Atomic Props to make children elements align to center.

| Atomic Props | Effect                                |
| ------------ | ------------------------------------- |
| toCenter     | Make children elements align to center |

## Usage

### toCenter with row

```jsx live
<View toCenter bgGray100 s-200>
  <View s-60 bgAmber400 rounded-8></View>
  <View s-80 bgBlue400 rounded-8></View>
</View>
```

### toCenter with column

```jsx live
<View column toCenter bgGray100 s-200>
  <View s-60 bgAmber400 rounded-8></View>
  <View s-80 bgBlue400 rounded-8></View>
</View>
```
