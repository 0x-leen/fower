title: To Center
---

Atomic Props to make children elements align to center.

| Atomic Props | Effect                                 |
| ------------ | -------------------------------------- |
| toCenter     | Make children elements align to center |

## Usage

### toCenter with row

```jsx live
<Box toCenter bgGray100 square-200>
  <Box square-60 bgOrange400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```

### toCenter with column

```jsx live
<Box column toCenter bgGray100 square-200>
  <Box square-60 bgOrange400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```
