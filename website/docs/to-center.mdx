import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic Props to make children elements align to center.

| Atomic Props | Effect                                 |
| ------------ | -------------------------------------- |
| toCenter     | Make children elements align to center |

## Usage

### toCenter with row

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box toCenter bgGray100 square-200>
  <Box square-60 bgOrange400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div toCenter bgGray100 square-200>
  <div square-60 bgOrange400 rounded-8></div>
  <div square-80 bgBlue400 rounded-8></div>
</div>
```

</TabItem>
</Tabs>
