title: Size
---

Use `s-{value}` props to controll element's width and height.

| Atomic Props          | CSS Properties                   |
| --------------------- | -------------------------------- |
| s-{value} / s={value} | width: {value}; height: {value}; |

## Usage

```jsx live
<View toLeft white>
  <View s-50 bgYellow400 toCenter>
    s-50
  </View>
  <View s-5rem bgGreen400 toCenter>
    s-5rem
  </View>
  <View s={100} bgBlue400 toCenter>
    s={'{100}'}
  </View>
  <View s="150px" bgPurple400 toCenter>
    s="150px"
  </View>
</View>
```
