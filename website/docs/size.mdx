title: Size
---

使用 `s-{size}` 格式的 Props 快速把元素变为正方形。

### 用法

```jsx live
<View toLeft>
  <View s-50 bgYellow200>
    Hello
  </View>
  <View s-5rem bgGreen200>
    Hello
  </View>
  <View s={100} bgBlue200>
    Hello
  </View>
  <View s="150px" bgPurple200>
    Hello
  </View>
</View>
```
