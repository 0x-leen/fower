## 使用设计系统的颜色

```jsx live
<View f-20>
  <Text gray60>王勃</Text>
  <Text colorPrimary> 李白</Text>
  <Text green40> 杜甫</Text>
</View>
```

所有的预设颜色：[Color palette](/docs/design-system-color)

## 使用 `color` props

```jsx live
<View f-20>
  <Text color="#999">王勃</Text>
  <Text color="deeppink"> 李白</Text>
  <Text color="#20B2AA"> 杜甫</Text>
</View>
```

## 速改变颜色透明度

```jsx live
<View row evenly>
  <View color="#ed8936">#ed8936</View>
  <View color="#ed8936.8">#ed8936.8</View>
  <View color="#ed8936.6">#ed8936.6</View>
  <View color="#ed8936.4">#ed8936.4</View>
  <View color="#ed8936.2">#ed8936.4</View>
</View>
```
