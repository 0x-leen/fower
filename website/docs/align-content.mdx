import Tabs from '@theme/Tabs'

Atomic props for controlling how rows are positioned in multi-row flex and grid containers.

| Atomic Props         | CSS Properties          |
| -------------------- | ----------------------- |
| alignContent={value} | align-content: {value}; |

## Start

Use `alignContent="flex-start"` to pack rows in a container against the start of the cross axis.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box bgOrange100 w-350 h-280 row alignContent="flex-start">
  <Box m2 bgOrange400 square-100></Box>
  <Box m2 bgOrange400 square-100></Box>
  <Box m2 bgOrange400 square-100></Box>
  <Box m2 bgOrange400 square-100></Box>
  <Box m2 bgOrange400 square-100></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div bgOrange100 w-350 h-280 row alignContent="flex-start">
  <div m2 bgOrange400 square-100></div>
  <div m2 bgOrange400 square-100></div>
  <div m2 bgOrange400 square-100></div>
  <div m2 bgOrange400 square-100></div>
  <div m2 bgOrange400 square-100></div>
</div>
```

</TabItem>
</Tabs>

## Space evenly

Use `alignContent="space-evenly"` to distribute rows in a container such that there is an equal amount of space around each item, but also accounting for the doubling of space you would normally see between each item when using content-around.

defaultValue="React"
