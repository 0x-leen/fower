import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic props for controlling how an element's overflow attribute.

| Atomic Props      | CSS Properties      |
| ----------------- | ------------------- |
| overflow={value}  | overflow: {value}   |
| overflowX={value} | overflow-x: {value} |
| overflowY={value} | overflow-y:{value}  |

## Overflow

Use `overflow` props to set an element's overflow.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box square-200 overflow="auto" bgGray100 p4>
  <Box square-400>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Assumenda, quia temporibus eveniet a
    libero incidunt suscipit laborum, rerum accusantium modi quidem, ipsam illum quis sed voluptatum
    quae eum fugit earum.
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div square-200 overflow="auto" bgGray100 p4>
  <div square-400>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Assumenda, quia temporibus eveniet a
    libero incidunt suscipit laborum, rerum accusantium modi quidem, ipsam illum quis sed voluptatum
    quae eum fugit earum.
  </div>
</div>
```

</TabItem>
</Tabs>
