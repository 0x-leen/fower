---

If you don't use React, Vue and other web frameworks, you can also install @styli/core and use Styli with the most primitive APIs.

```bash
$ npm i @styli/core @styli/preset-default
```

## Basic Usage

```jsx
import { styli, css } from '@styli/core'
import preset from '@styli/preset-default'

styli.configure(preset)

const className = css('p-20 mx-10 borderGray-1 f-40')

const $app = document.getElementById('root')

$app.className = className
```

你也可以增加额外的自定义样式：

## Wich custom style

```jsx
import { css } from '@styli/core'

const className = css('p-20 mx-10 borderGray-1 f-40', {
  display: 'inline-block',
  textShadow: '1px 1px 2px pink',
})

const $app = document.getElementById('root')

$app.className = className
```
