---

If you don't use React and other web frameworks, you can also install `@styli/core` and use Styli with the most primitive APIs.

```bash
$ npm i @styli/core @styli/preset-web
```

## Basic Usage

```jsx
import { styli, css } from '@styli/core'
import preset from '@styli/preset-web'

styli.configure(preset)

const className = css('p-20 mx-10 borderGray text4XL')

const $app = document.getElementById('root')

$app.className = className
```

## With custom style

```jsx
import { css } from '@styli/core'

const className = css('p-20 mx-10 borderGray text4XL', {
  display: 'inline-block',
  textShadow: '1px 1px 2px pink',
})

const $app = document.getElementById('root')

$app.className = className
```
