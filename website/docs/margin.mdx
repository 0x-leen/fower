Control the margin of an element using the `m{t|r|b|l|x|y}-{size}` Atomic Props.

## Margin Props

`margin` is a style attribute used frequently, so we use abbreviations.

- `m` margin
- `mt` margin-top
- `mr` margin-right
- `mb` margin-bottom
- `ml` margin-left
- `mx` margin-left and margin-right
- `my` margin-top and margin-bottom

### Margin of all sides

Control the margin on all sides of an element using the `m-{size}` Atomic Props.

```jsx live
<View bgGray10 left>
  <View bgGreen20 s-100 m-0>
    Hello
  </View>
  <View bgGreen20 s-100 m={20}>
    Hello
  </View>
  <View bgGreen20 s-100 m="40px">
    Hello
  </View>
  <View bgGreen20 s-100 m="10px 20px">
    Hello
  </View>
</View>
```

### Margin of one side

Control the margin on one side of an element using the `m{t|r|b|l}-{size}` Atomic Props.

```jsx live
<View bgGray10 left>
  <View ml-10 bgGreen20 s-100>
    Hello
  </View>
  <View mt-10 bgTeal20 s-100>
    Hello
  </View>
  <View mr-10 bgRed20 s-100>
    Hello
  </View>
  <View mb-10 bgYellow20 s-100>
    Hello
  </View>
</View>
```

### Horizontal margin

Control the horizontal margin of an element using the `mx-{size}` Atomic Props.

```jsx live
<View bgGray10 left>
  <View mx-10 s-100 bgRed20>
    Hello
  </View>
</View>
```

### Vertical marign

Control the vertical margin of an element using the `my-{size}` Atomic Props.

```jsx live
<View bgGray10 left>
  <View my-10 s-100 bgRed20>
    Hello
  </View>
</View>
```

### 设置负 margin

Control the negative margin of an element using the `m{t|r|b|l|x|y}--{size}` Atomic Props.

Of course, you can also use `ml={-10}` to set negative margin of an element.

```jsx live
<View p-50 debugChildren>
  <View ml--10>Hello</View>
  <View mr--10px>Hello</View>
  <View mt--1rem>Hello</View>
  <View mb={-10}>Hello</View>
  <View ml="-10px">Hello</View>
  <View mr="-1rem">Hello</View>
</View>
```
