title: List Type
---

Use `h-{value}` props to controll element height.

| Atomic Props            | CSS Properties                          |
| ----------------------- | --------------------------------------- |
| listNone                | list-style-type: none;                  |
| listDisc                | list-style-type: disc;                  |
| listDecimal             | list-style-type: decimal;               |
| listSquare              | list-style-type: square;                |
| listCircle              | list-style-type: circle;                |
| listTradChineseInformal | list-style-type: trad-chinese-informal; |
| listLowerRoman          | list-style-type: lower-roman;           |
| listUpperRoman          | list-style-type: upper-roman;           |
| listLowerAlpha          | list-style-type: lower-alpha;           |
| listUpperAlpha          | list-style-type: upper-alpha;           |
