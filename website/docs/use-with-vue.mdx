## 安装

```bash
npm i @styli/core @styli/vue @styli/preset-default
```

注意，除了需要 `@styli/core`，你还需安装和 Vue 对应的组件包 `@styli/vue`。

## 初始化 Preset

在使用 Styli 编写样式之前，你需要通过 `styli.setup(preset)` 初始化插件。

```js title="src/index.ts"
import { styli } from '@styli/core'
import preset from '@styli/preset-default'

styli.setup(preset)
```

## 使用

首先，你需要注册 Styli 的 Vue 插件:

```jsx
import Vue from 'vue'
import styliVue from '@styli/vue'

Vue.use(styliVue)
```

然后怎么使用呢? Styli 在 Vue 有两种使用方式：

- 使用 `v-css` 指令
- 使用自定义组件

### 使用 `v-css` 指令

使用指令的方式，你可以给 `v-css` 传递你想要的 modifiers：

```jsx
<template>
  <div v-css="'centerY p-10 w-260 bgWhite rounded-10 shadow'">
    <img v-css="'c-48'" src="/img/jobs.jpg" />
    <div v-css="'ml-10'">
      <div v-css="'f-20 fontBold'">Steve Jobs</div>
      <span v-css="'gray80'">Co-founder of Apple Inc.</span>
    </div>
  </div>
</template>
```

你也可以不给 `v-css` 传值，通过给 html 元素添加属性实现样式:

```jsx
<template>
  <div v-css centerY p-10 w-260 bgWhite rounded-10 shadow>
    <img v-css c-48 src="/img/jobs.jpg" />
    <div v-css ml-10>
      <div v-css f-20 fontBold>
        Steve Jobs
      </div>
      <span v-css gray80>
        Co-founder of Apple Inc.
      </span>
    </div>
  </div>
</template>
```

### 使用自定义组件

你可以使用 `@styli/vue` 提供的自定义组件，例如 `Div`、`Image`、`Span`:

```jsx
<template>
  <Div centerY p-10 w-260 bgWhite rounded-10 shadow>
    <Image c-48 src="/img/jobs.jpg" />
    <Div ml-10>
      <Div f-20 fontBold>
        Steve Jobs
      </Div>
      <Span gray80>Co-founder of Apple Inc.</Span>
    </Div>
  </Div>
</template>
```

两种方式怎么选择呢？如果你想在的代码中保留传统的 html 标签，你可以使用指令的方式；如果你是 React 用户，你也许会喜欢自定义组件的方式。
