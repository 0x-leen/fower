## Installation

```bash
npm i @fower/vue
```

## Basic usage

First, you need to register Fower's Vue plugin:

```jsx
import Vue from 'vue'
import fowerVue from '@fower/vue'

Vue.use(fowerVue)
```

Then, add the `v-css` directive to the element, and then add Atomic Prop to create the style:

```jsx
<template>
  <div v-css toCenterY p-10 w-260 bgWhite rounded-10 shadow>
    <img v-css circle-48 src="/img/jobs.jpg" />
    <div v-css ml-10>
      <div v-css textXL fontBold>
        Steve Jobs
      </div>
      <span v-css gray800>
        Co-founder of Apple Inc.
      </span>
    </div>
  </div>
</template>
```

## Advanced usage

In the above usage, when we add Atomic Props to HTML elements, we also need to add a `v-css` directive, which is particularly cumbersome. Can we not add the `v-css` directive? The answer is yes.

How to use it? You can modify the vue-loader configuration in the webpack configuration to get it. For example, if you are using Vue CLI, modify `vue.config.js`.

```bash
npm i @fower/vue-template-compiler
```

```js title="vue.config.js"
module.exports = {
  parallel: false,
  chainWebpack(config) {
    config.module
      .rule('vue')
      .use('vue-loader')
      .tap((options) => {
        options.compiler = require('@fower/vue-template-compiler')
        return options
      })
  },
}
```

Then you don't need to write the `v-css` directive, but write Atomic Props directly in HTML:

Note: use with vue-cli, you should set `parallel: false` in `vue.config.js`.

```jsx
<template>
  <div toCenterY p-10 w-260 bgWhite rounded-10 shadow>
    <img circle-48 src="/img/jobs.jpg" />
    <div ml-10>
      <div textXL fontBold>
        Steve Jobs
      </div>
      <span gray800>Co-founder of Apple Inc.</span>
    </div>
  </div>
</template>
```
