## Installation

```bash
npm i @styli/core @styli/vue @styli/preset-default
```

Besides `@styli/core`, we also need React binding package `@styli/vue`.

## Init Preset

Before write styles with Styli, you need to initialize the plugins via `styli.configure(preset)`.

```js title="src/index.ts"
import { styli } from '@styli/core'
import preset from '@styli/preset-default'

styli.configure(preset)
```

## Basic usage

First, you need to register Styli's Vue plugin:

```jsx
import Vue from 'vue'
import styliVue from '@styli/vue'

Vue.use(styliVue)
```

Then, add the `v-css` directive to the element, and then add Atomic Prop to create the style:

```jsx
<template>
  <div v-css centerY p-10 w-260 bgWhite rounded-10 shadow>
    <img v-csscircle-48 src="/img/jobs.jpg" />
    <div v-css ml-10>
      <div v-css f-20 fontBold>
        Steve Jobs
      </div>
      <span v-css gray80>
        Co-founder of Apple Inc.
      </span>
    </div>
  </div>
</template>
```

## Advanced usage

In the above usage, when we add Atomic Props to HTML elements, we also need to add a `v-css` directive, which is particularly cumbersome. Can we not add the `v-css` directive? The answer is yes.

How to use it? You can modify the vue-loader configuration in the webpack configuration to get it. For example, if you are using Vue CLI, modify `vue.config.js`.

```bash
npm i @styli/vue-template-compiler
```

```js title="vue.config.js"
module.exports = {
  lintOnSave: false,
  chainWebpack(config) {
    config.module
      .rule('vue')
      .use('vue-loader')
      .tap((options) => {
        options.compiler = require('@styli/vue-template-compiler')
        return options
      })
  },
}
```

Then you don't need to write the `v-css` directive, but write Atomic Props directly in HTML:

```jsx
<template>
  <div centerY p-10 w-260 bgWhite rounded-10 shadow>
    <imgcircle-48 src="/img/jobs.jpg" />
    <div ml-10>
      <div f-20 fontBold>
        Steve Jobs
      </div>
      <span gray80>Co-founder of Apple Inc.</span>
    </div>
  </div>
</template>
```
