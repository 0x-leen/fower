id: styled
title: Styled Components
---

You can use `styled` api to create a Fower component, then you can use atomic props with the component.

:::info
`styled` can be used in all React-like frameworks, such as React, React-native, Preact, etc.
:::

## Import

```jsx
import { styled } from '@fower/styled'
```

## Styling html element

The element wrapped by `styled` can use atomic props to change the style of the component.

```jsx live
function StyledElement() {
  const Text = styled('span')
  return (
    <Text text3XL fontBold green400>
      Hello world
    </Text>
  )
}
```

## Styling any Component

`styled` can style any component as long as it accepts a className prop.

```jsx live
function StyledAnyComponent() {
  const Button = ({ className }) => <button className={className}>Styled Button</button>
  const StyledButton = styled(Button)

  return <StyledButton textLG white bgGreen400 p-10 rounded-4 border-0 />
}
```

## Reuse style

为了更好地复用样式，使用`styled`创建组件时，我们可以给组件初始化一些样式。

使用 modifier：

```js
const StyledItem = styled(Item, 'p-10 bgRed white text3XL')
```

如果 modifier 不能满足，你可以使用 CSS Object style：

```js
const StyledItem = styled(Item, {
  display: 'inline-block',
  lineHeight: '30px',
})
```

或者两种并存：

```js
const StyledItem = styled(Item, 'p-10 bgRed white text3XL', {
  display: 'inline-block',
  lineHeight: '30px',
})
```

或者接受多个参数:

```js
const StyledItem = styled(Item, 'p-10', 'bgRed', 'white', 'text3XL', {
  display: 'inline-block',
  lineHeight: '30px',
})
```
