## Font Size

Control the font size by `f-{size}` Atomic Props.

:::info
The `fontSize` is a frequently used style, so we simplified it to `f` a letter.
:::

```jsx live
<View column>
  <Text f-14>Lorem ipsum dolor sit amet</Text>
  <Text f={20}>Lorem ipsum dolor sit amet</Text>
  <Text f-2rem>Lorem ipsum dolor sit amet</Text>
</View>
```

## Font Weight

There are two ways to control the font weight, one is to use a preset value, and the other is to use a precise value.

Use preset value:

```jsx live
<View f-28>
  <View fontHairline>Lorem ipsum dolor sit amet</View>
  <View fontThin>Lorem ipsum dolor sit amet</View>
  <View fontLight>Lorem ipsum dolor sit amet</View>
  <View fontNormal>Lorem ipsum dolor sit amet</View>
  <View fontMedium>Lorem ipsum dolor sit amet</View>
  <View fontSemibold>Lorem ipsum dolor sit amet</View>
  <View fontBold>Lorem ipsum dolor sit amet</View>
  <View fontExtrabold>Lorem ipsum dolor sit amet</View>
</View>
```

Use precise value:

```jsx live
<View f-24>
  <View fontWeight-100>Lorem ipsum dolor sit amet</View>
  <View fontWeight-400>Lorem ipsum dolor sit amet</View>
  <View fontWeight={600}>Lorem ipsum dolor sit amet</View>
</View>
```

## Line Height

There are two ways to control the line height of the text, one is to use a preset value, and the other is to use a precise pass value.

Use preset value: `lhNone`、`lhTight`、`lhSnug`、`lhNormal`、`lhRelaxed`、`lhLoose`.

<LineHeight />

```jsx
<View>
  <View lhNone>Lorem ipsum dolor sit amet...</View>
  <View lhTight>Lorem ipsum dolor sit amet...</View>
  <View lhSnug>Lorem ipsum dolor sit amet...</View>
  <View lhNormal>Lorem ipsum dolor sit amet...</View>
  <View lhRelaxed>Lorem ipsum dolor sit amet...</View>
  <View lhLoose>Lorem ipsum dolor sit amet...</View>
</View>
```

Use precise value:

```jsx
<View f-20>
  <View lh-20>Lorem ipsum dolor sit amet...</View>
  <View lh-3em>Lorem ipsum dolor sit amet...</View>
  <View lh="1.5em">Lorem ipsum dolor sit amet...</View>
</View>
```

## Text Alignment

Use `textLeft`, `textCenter`, `textRight`, `textJustify` to control the alignment of the text.

```jsx live
<View f-20>
  <View textLeft>Lorem ipsum dolor sit amet</View>
  <View textCenter>Lorem ipsum dolor sit amet</View>
  <View textRight>Lorem ipsum dolor sit amet</View>
  <View textJustify>Lorem ipsum dolor sit amet</View>
</View>
```

## Text Transform

Use `normalcase`, `uppercase`, `lowercase`, `capitalize` to control the Text Transform.

```jsx live
<View f-20>
  <View uppercase>Lorem ipsum dolor sit amet</View>
  <View lowercase>Lorem ipsum dolor sit amet</View>
  <View capitalize>Lorem ipsum dolor sit amet</View>
  <View normalcase>Lorem ipsum dolor sit amet</View>
</View>
```
