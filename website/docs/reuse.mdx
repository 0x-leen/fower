When a component uses a lot of Atomic Props, and this style will be used by multiple components, it is recommended to create a custom component.

## Wrap Fower Component

```jsx
import React from 'react'
import { Button } from '@fower/react'

export const MyButton = (props) => {
  return <Button {...props} p-10 textLG bgYello400 white rounded-8 />
}
```

## Styled HTML element

```jsx
import React from 'react'
import { Button } from '@fower/react'

export const MyButton = styled('button', 'p-10', 'textLG', 'bgYello400', 'rounded-8', {
  textShadow: '1px 0 5px #FC0',
})
```

## Styled thirdparty Component

```jsx
import { styled } from '@fower/styled'
import { Button } from 'antd'

const StyledButton = styled(Button)

function Demo() {
  return (
    <div>
      <StyledButton p-10>click me</StyledButton>
    </div>
  )
}
```

**Note:** The component wrapped by `styled` must have the className Props.
