---
id: min-width
title: Min Width
---

Use `minW-{value}` Atomic Props to control the min width of an element.

| Atomic Props                | CSS Properties      |
| --------------------------- | ------------------- |
| minW-{value} / minW={value} | min-width: {value}; |

## Fixed Max-Width

Use `minW-{size}` Atomic Props to control the width of an element.

```jsx live
<View white>
  <View minW-100 bgRed400 inlineBlock p2>
    minW-100
  </View>
  <View minW-10rem bgYellow400 inlineBlock p2>
    minW-10rem
  </View>
  <View minW={100} bgGreen400 inlineBlock p2>
    minW={'{100}'}
  </View>
  <View minW="400px" bgBlue400 inlineBlock p2>
    minW="400px"
  </View>
  <View minW="80%" bgPurple400 inlineBlock p2>
    minW="80%"
  </View>
</View>
```

## Fluid Max-Width

```jsx live
<View white textCenter spaceY2>
  <View toLeft>
    <View minW={1 / 2} bgOrange400 py-10>
      minW={'{1 / 2}'}
    </View>
    <View minW={1 / 2} bgRed400 py-10>
      minW={'{1 / 2}'}
    </View>
  </View>

  <View toLeft>
    <View minW={1 / 3} bgOrange400 py-10>
      minW={'{1 / 3}'}
    </View>
    <View minW={2 / 3} bgRed400 py-10>
      minW={'{2 / 3}'}
    </View>
  </View>

  <View toLeft>
    <View minW={1 / 4} bgOrange400 py-10>
      minW={'{1 / 4}'}
    </View>
    <View minW={3 / 4} bgRed400 py-10>
      minW={'{3 / 4}'}
    </View>
  </View>

  <View toLeft>
    <View minW={0.5} bgOrange400 py-10>
      minW={'{0.5}'}
    </View>
    <View minW={0.5} bgRed400 py-10>
      minW={'{0.5}'}
    </View>
  </View>
</View>
```
