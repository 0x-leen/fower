Use atomic props to control the width of an element. width is a property used frequently, so we use abbreviations.

## Fixed width

Use `w-{size}` Atomic Props to control the width of an element.

```jsx live
<View>
  <View w-100 bgRed20>
    Hello
  </View>
  <View w-10rem bgYellow20>
    Hello
  </View>
  <View w={50} bgGreen20>
    Hello
  </View>
  <View w="200px" bgBlue20>
    Hello
  </View>
  <View w="80%" bgPurple20>
    Hello
  </View>
  <View w={1 / 3} bgTeal20>
    Hello
  </View>
</View>
```

## Min-width

Use `minW-{size}` Atomic Props to control the min width of an element.

```jsx live
<View minW-100 bgRed20 inlineBlock>
  Hello
</View>
```

## Max-width

Use `maxW-{size}` Atomic Props to control the max width of an element.

```jsx live
<View maxW-100 bgRed20>
  Hello
</View>
```
