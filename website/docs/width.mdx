Use atomic props to control the width of an element. width is a property used frequently, so we use abbreviations.

| Atomic Props          | CSS Properties  |
| --------------------- | --------------- |
| w-{value} / w={value} | width: {value}; |

## Fixed width

Use `w-{size}` Atomic Props to control the width of an element.

```jsx live
<View white>
  <View w-100 bgRed400 p2>
    w-100
  </View>
  <View w-10rem bgYellow400 p2>
    w-10rem
  </View>
  <View w={100} bgGreen400 p2>
    w={'{100}'}
  </View>
  <View w="400px" bgBlue400 p2>
    w="400px"
  </View>
  <View w="80%" bgPurple400 p2>
    w="80%"
  </View>
</View>
```

## Fluid Width

```jsx live
<View white textCenter spaceY2>
  <View toLeft>
    <View w={1 / 2} bgOrange400 py-10>
      w={'{1 / 2}'}
    </View>
    <View w={1 / 2} bgRed400 py-10>
      w={'{1 / 2}'}
    </View>
  </View>

  <View toLeft>
    <View w={1 / 3} bgOrange400 py-10>
      w={'{1 / 3}'}
    </View>
    <View w={2 / 3} bgRed400 py-10>
      w={'{2 / 3}'}
    </View>
  </View>

  <View toLeft>
    <View w={1 / 4} bgOrange400 py-10>
      w={'{1 / 4}'}
    </View>
    <View w={3 / 4} bgRed400 py-10>
      w={'{3 / 4}'}
    </View>
  </View>

  <View toLeft>
    <View w={0.1} bgOrange400 py-10>
      w={'{0.1}'}
    </View>
    <View w={0.9} bgRed400 py-10>
      w={'{0.9}'}
    </View>
  </View>
</View>
```
