使用原子 Props 控制元素的宽度。

宽度是高频率使用的属性，所以我们使用缩写。

### 固定宽度

使用 w-{size} 格式的 Props 可以控制元素的宽度。

```jsx
<View w-50>Hello</View>
<View w-50rem>Hello</View>
<View w={50}>Hello</View>
<View w="50px">Hello</View>
<View w="100%">Hello</View>
```

### 最小宽度

使用 minW-{size} 格式的 Props 可以控制元素的最小宽度。

```jsx
<View minW-50>Hello</View>
<View minW-50rem>Hello</View>
<View minW={50}>Hello</View>
<View minW="50px">Hello</View>
<View minW="100%">Hello</View>
```

### 最大宽度

使用 maxW-{size} 格式的 Props 可以控制元素的最大宽度。

```jsx
<View maxW-50>Hello</View>
<View maxW-50rem>Hello</View>
<View maxW={50}>Hello</View>
<View maxW="50px">Hello</View>
<View maxW="100%">Hello</View>
```
