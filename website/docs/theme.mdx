The theme config is where you define your design system like
color palette, spacing, fonts, border radius, breakpoints, etc.

## Theme structure

    breakpoints: {
      sm: '640px',

      lg: '1024px',
      xl: '1280px',
      '2xl': '1536px',

    spacings: {

      14: 56,
      16: 64,
      20: 80,
      24: 96,
      28: 112,
