import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic props for controlling the placement of positioned elements.

| Atomic Props                    | CSS Properties |
| ------------------------------- | -------------- |
| top-{value} / top={value}       | top={value}    |
| right-{value} / right={value}   | right={value}  |
| bottom-{value} / bottom={value} | bottom={value} |
| left-{value} / left={value}     | left={value}   |

## Top

Use `top` props to set an element's top position.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box relative square-160 bgOrange100>
  <Box bgOrange400 square-80 absolute top-20></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div relative square-160 bgOrange100>
  <div bgOrange400 square-80 absolute top-20></div>
</div>
```

</TabItem>
</Tabs>
