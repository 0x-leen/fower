## 使用

Fower 是跨框架的，下面我们看看在 React 的用法，使用 Fower 提供的 `Box` 组件，你可以快速实现复杂界面。

<GettingStarted />

```jsx
import React from 'react'
import { Box } from '@fower/react'

export default () => {
  return (
    <Box toCenterY p-10 w-260 bgWhite rounded-10>
      <Box as="img" circle-48 src="/img/jobs.jpg" />
      <Box ml-10>
        <Box textXL fontBold>
          Steve Jobs
        </Box>
        <Text gray800>Co-founder of Apple Inc.</Text>
      </Box>
    </Box>
  )
}
```

Fower 巧妙地利用了 JSX 的 boolean(true) 类型 props 可以省略值的特性，实现了 "Key as Value" 模式，让 Fower 在是构建界面时更加简洁，同时由于使用了 tsx，我们得到了强类型的 "CSS in JS"。
