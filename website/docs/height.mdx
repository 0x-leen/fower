使用原子 Props 控制元素的高度。

高度是高频率使用的属性，所以我们使用缩写。

### 固定高度

使用 h-{size} 格式的 Props 可以控制元素的高度。

```jsx
<View h-50>Hello</View>
<View h-50rem>Hello</View>
<View h={50}>Hello</View>
<View h="50px">Hello</View>
<View h="100%">Hello</View>
```

### 最小宽度

使用 minH-{size} 格式的 Props 可以控制元素的最小高度。

```jsx
<View minH-50>Hello</View>
<View minH-50rem>Hello</View>
<View minH={50}>Hello</View>
<View minH="50px">Hello</View>
<View minH="100%">Hello</View>
```

### 最大宽度

使用 maxH-{size} 格式的 Props 可以控制元素的最大高度。

```jsx
<View maxH-50>Hello</View>
<View maxH-50rem>Hello</View>
<View maxH={50}>Hello</View>
<View maxH="50px">Hello</View>
<View maxH="100%">Hello</View>
```
