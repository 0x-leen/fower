title: Height
---

Use `h-{value}` props to controll element height.

| Atomic Props          | CSS Properties   |
| --------------------- | ---------------- |
| h-{value} / h={value} | height: {value}; |

## Usage

```jsx live
<View toLeft white h-100 bgGray100>
  <View h-50 bgRed400 p2>
    h-50
  </View>
  <View h-5rem bgYellow400 p2>
    h-5rem
  </View>
  <View h={50} bgGreen400 p2>
    h={'{50}'}
  </View>
  <View h="100px" bgBlue400 p2>
    h="100px"
  </View>
  <View h="80%" bgPurple400 p2>
    h="80%"
  </View>
</View>
```
