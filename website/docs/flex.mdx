import Tabs from '@theme/Tabs'

| Atomic Props | CSS Properties  |
| ------------ | --------------- |
| flexAuto     | flex: 1 1 auto; |
| flexInitial  | flex: 0 1 auto; |
| flexNone     | flex: 'none'    |
| flex={value} | flex: {value}   |

## Flex Initial

Use `flexInitial` props to allow a flex item to shrink but not grow, taking into account its initial size.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box row spaceX3 flexInitial>
  <Box bgAmber400 p10>
    Short
  </Box>
  <Box bgAmber400 p10>
    Medium length
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div row spaceX3 flexInitial>
  <div bgAmber400 p10>Short</div>
  <div bgAmber400 p10>Medium length</div>
</div>
```

</TabItem>
</Tabs>

## Flex None

Use `flexNone` props to prevent a flex item from growing or shrinking.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box row spaceX3 flexInitial>
  <Box bgAmber400 p4 flex={1}>
    Item that can grow or shrink if needed
  </Box>
  <Box bgAmber500 p4 flexNone>
    Item that cannot grow or shrink
  </Box>
  <Box bgAmber400 p4 flex="1">
    Item that can grow or shrink if needed
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div row spaceX3 flexInitial>
  <div bgAmber400 p4 flex="1">Item that can grow or shrink if needed</div>
  <div bgAmber500 p4 flexNone>Item that cannot grow or shrink</div>
  <div bgAmber400 p4 flex="1">Item that can grow or shrink if needed</div>
</div>
```

</TabItem>
</Tabs>

## Any Value

Use `flex={value}` to set any flex value.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box row spaceX3 row>
  <Box bgAmber400 p4 flex={1}></Box>
  <Box bgAmber500 p4 flex={2}></Box>
  <Box bgAmber400 p4 flex={1}></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div row spaceX3 row>
  <div bgAmber400 p4 flex="1"></div>
  <div bgAmber500 p4 flex="2"></div>
  <div bgAmber400 p4 flex="1"></div>
</div>
```

</TabItem>
</Tabs>
