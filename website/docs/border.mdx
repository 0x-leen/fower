title: Border
---

使用 `border{T|R|B|L|X|Y}` 格式的 Props 可以控制元素的 border。

## 四周边框

```jsx live
<View s-80 bgGray200 border="3px solid deeppink">
  border
</View>
```

## 单一方向

```jsx live
<View row toEvenly>
  <View s-80 bgGray200 borderT="3px solid deeppink">
    borderT
  </View>
  <View s-80 bgGray200 borderR="3px solid deeppink">
    borderR
  </View>
  <View s-80 bgGray200 borderB="3px solid deeppink">
    borderB
  </View>

  <View s-80 bgGray200 borderL="3px solid deeppink">
    borderL
  </View>
</View>
```

## 使用伪类

你可以使用 styli 支持的所有伪类，这里以 hover 和 active 举例。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<View toEvenly>
  <View s-80 bgGray200 border="3px solid deeppink" border--hover="3px solid yellowGreen">
    hover
  </View>
  <View s-80 bgGray200 border="3px solid deeppink" border--active="3px solid yellowGreen">
    active
  </View>
</View>
```
