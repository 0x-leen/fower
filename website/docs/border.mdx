id: border
title: Border
---

使用 `border{T|R|B|L|X|Y}` 格式的 Props 可以控制元素的 border。

## 四周边框

```jsx live
<View s-80 bgGray20 border="3px solid deeppink">
  border
</View>
```

## 单一方向

```jsx live
<View row evenly>
  <View s-80 bgGray20 borderT="3px solid deeppink">
    borderT
  </View>
  <View s-80 bgGray20 borderR="3px solid deeppink">
    borderR
  </View>
  <View s-80 bgGray20 borderB="3px solid deeppink">
    borderB
  </View>

  <View s-80 bgGray20 borderL="3px solid deeppink">
    borderL
  </View>
</View>
```

## X、Y 方向

```jsx live
<View row evenly>
  <View s-80 bgGray20 borderX="3px solid deeppink">
    borderX
  </View>
  <View s-80 bgGray20 borderY="3px solid deeppink">
    bordeY
  </View>
</View>
```
