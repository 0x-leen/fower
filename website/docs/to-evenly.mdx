Atomic Props to make children elements evenly.

| Atomic Props | Effect                        |
| ------------ | ----------------------------- |
| toEvenly     | Make children elements evenly |

## Usage

Use `toEvenly` to make children elements evenly.

```jsx live
<Box toEvenly bgGray100 s-200>
  <Box s-60 bgAmber400 rounded-8></Box>
  <Box s-80 bgBlue400 rounded-8></Box>
</Box>
```
