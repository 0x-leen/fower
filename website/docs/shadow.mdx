title: Shadow
---

Use `shadow{SIZE}` Atomic Props controll element's shadow.

| Atomic Props   | CSS Properties         |
| -------------- | ---------------------- |
| shadowTiny     | box-shadow: {tiny}     |
| shadowSmall    | box-shadow: {small}    |
| shadowMedium   | box-shadow: {medium}   |
| shadowLarge    | box-shadow: {large}    |
| shadowHuge     | box-shadow: {huge}     |
| shadowGigantic | box-shadow: {gigantic} |
| shadow={value} | box-shadow: {value}    |

## Usage

```jsx live
<Box bgGray100 p-20 toCenterX toBetween>
  <Box toCenter square-60 bgWhite shadowTiny>
    tiny
  </Box>
  <Box toCenter square-80 bgWhite bgWhite shadowSmall>
    small
  </Box>
  <Box toCenter square-80 bgWhite shadowMedium>
    medium
  </Box>
  <Box toCenter square-90 bgWhite shadowLarge>
    large
  </Box>
  <Box toCenter square-100 bgWhite shadowHuge>
    huge
  </Box>
  <Box toCenter square-110 bgWhite shadowGigantic>
    gigantic
  </Box>
</Box>
```

## Default shadow

```jsx live
<Box bgGray100 p-20 toCenter>
  <Box toCenter square-100 bgWhite shadow>
    default
  </Box>
</Box>
```
