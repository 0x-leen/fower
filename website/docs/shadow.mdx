import Tabs from '@theme/Tabs'

Use `shadow{SIZE}` Atomic Props controll element's shadow.

| Atomic Props   | CSS Properties      |
| -------------- | ------------------- |
| shadowSM       | box-shadow: {SM}    |
| shadow         | box-shadow: {Base}  |
| shadowMD       | box-shadow: {MD}    |
| shadowLG       | box-shadow: {LG}    |
| shadowXL       | box-shadow: {XL}    |
| shadow2XL      | box-shadow: {2XL}   |
| shadow={value} | box-shadow: {value} |

## Usage

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box bgGray100 p-20 toCenterX toBetween>
  <Box toCenter square-60 bgWhite shadowSM>
    SM
  </Box>
  <Box toCenter square-80 bgWhite bgWhite shadow>
    Base
  </Box>
  <Box toCenter square-80 bgWhite shadowMD>
    MD
  </Box>
  <Box toCenter square-90 bgWhite shadowLG>
    LG
  </Box>
  <Box toCenter square-100 bgWhite shadowXL>
    XL
  </Box>
  <Box toCenter square-110 bgWhite shadow2XL>
    2XL
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div bgGray100 p-20 toCenterX toBetween>
  <div toCenter square-60 bgWhite shadowSM>SM</div>
  <div toCenter square-80 bgWhite bgWhite shadow>Base</div>
  <div toCenter square-80 bgWhite shadowMD>MD</div>
  <div toCenter square-90 bgWhite shadowLG>LG</div>
  <div toCenter square-100 bgWhite shadowXL>XL</div>
  <div toCenter square-110 bgWhite shadow2XL>2XL</div>
</div>
```

</TabItem>
</Tabs>
