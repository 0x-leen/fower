Atomic Props to make children elements evenly.

| Atomic Props | Effect                        |
| ------------ | ----------------------------- |
| toEvenly     | Make children elements evenly |

## Usage

Use `toEvenly` to make children elements evenly.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box toEvenly bgGray100 square-200>
  <Box square-60 bgAmber400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```jsx live
<div toEvenly bgGray100 square-200>
  <div square-60 bgAmber400 rounded-8></div>
  <div square-80 bgBlue400 rounded-8></div>
</div>
```

</TabItem>
</Tabs>
