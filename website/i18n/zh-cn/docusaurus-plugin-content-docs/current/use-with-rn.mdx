[React Native](https://reactnative.dev/) 是一个使用 React 来创建 Android 和 iOS 的原生应用框架。

:::note
待验证、不能用于生产环境
:::

## 安装

```bash
npm i @styli/react-native -S
```

## 使用

从 **@styli/react-native** 包中导入组件

```tsx
import { View, Image, Text } from '@styli/react-native'
```

使用组件，实现一个用户信息卡片界面。

```jsx live
<View centerY p-10 w-260 bgWhite rounded-10 shadow>
  <Image circle-48 src="/img/jobs.jpg" />
  <View ml-10>
    <View f-20 fontBold>
      Steve Jobs
    </View>
    <Text gray80>Co-founder of Apple Inc.</Text>
  </View>
</View>
```
