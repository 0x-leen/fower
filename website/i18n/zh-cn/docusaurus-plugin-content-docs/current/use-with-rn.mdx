在 React Native 中使用 Styli 与在 React 中使用 Styli 非常相似。

## 安装

```bash
npm i @styli/react-native
```

## 用法

下面，我们使用组件，实现一个用户信息卡片界面:

```jsx
import React from 'react'
import { View, Text, Image } from '@styli/react-native'

const ProfileCard = () => (
  <View toCenterY p-10 w-260 bgWhite rounded-10 shadow>
    <Image circle-48 source={{ uri: '/img/jobs.jpg' }} />
    <View ml-10>
      <View textXL fontBold>
        Steve Jobs
      </View>
      <Text gray800>Co-founder of Apple Inc.</Text>
    </View>
  </View>
)
```
