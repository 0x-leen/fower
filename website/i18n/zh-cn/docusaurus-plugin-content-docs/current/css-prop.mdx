大部分情况，你用原子属性可以完成 UI 开发，但有时候 Styli 提供的原子属性并不能满足的需求。因为 Styli 的原子属性只是 CSS 的子集，我们不会把所有 CSS 功能整合到 Styli 中。Styli 有一特殊的原子属性，叫 `css` 属性，它支持所有 CSS 功能。

## 基本用法

`css` 属性可以接收一个 CSS 对象，里面你可以写任何 CSS 属性，一些 Styli 原子属性不支持的冷门样式属性，你可以写在 `css` 属性中。

```jsx live
<Text
  f-40
  red500
  css={{
    textShadow: '1px 0 5px #FC0',
    textDecoration: 'underline',
  }}
>
  Hi, Styli!
</Text>
```

:::info
CSS 对象只能在 React-like 框架中使用，Vue.js 中无法使用。
:::

## 嵌套选择器

有时，你可能要使用 CSS 嵌套选择器，Styli 支持和 Less/Sass/Stylus 等类似的嵌套语法，只不过 Styli 是更结构化更灵活的 JS 对象，而不是字符串。

```jsx live
<View
  f-40
  css={{
    backgroundColor: '#edf2f7',
    ':hover': {
      backgroundColor: '#feebc8',
    },
    '.title': {
      fontWeight: 'bold',
    },
    span: {
      color: 'deeppink',
    },
  }}
>
  <div className="title">Nested demo</div>
  <span>Hi, Styli!</span>
</View>
```
