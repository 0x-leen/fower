title: ellipsis 文字省略
---

使用 `ellipsis{lineNum}-{maxWidth}` 格式的属性，可以快速设置文本省略。

## 单行文本省略

`ellipsis` 属性默认行数为 1，默认最大宽度为 100

```tsx live
<Wrapper>
  <Box h-100 w-200 bgBlue300>
    <Box ellipsis>Styli.Styli.Styli.Styli</Box>
  </Box>
  <Box h-100 w-200 bgBlue300>
    <Box ellipsis1-100>Styli.Styli.Styli.Styli</Box>
  </Box>
  <Box h-100 w-200 bgBlue300>
    <Box ellipsis-150>Styli.Styli.Styli.Styli</Box>
  </Box>
</Wrapper>
```

## 多行文本省略

```tsx live
<Wrapper>
  <Box h-100 w-200 bgBlue300>
    <Box ellipsis2>Styli.Styli.Styli.Styli.Styli.Styli.Styli.Styli</Box>
  </Box>
  <Box h-100 w-200 bgBlue300>
    <Box ellipsis3-150>
      Styli.Styli.Styli.Styli.Styli.Styli.Styli.Styli.Styli.Styli.Styli.Styli
    </Box>
  </Box>
</Wrapper>
```

**注意: 多行文本省略使用了实验性质的 [box-orient](https://developer.mozilla.org/zh-CN/docs/Web/CSS/box-orient)，请酌情使用 **
