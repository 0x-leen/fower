使用 `display` 可快速设置元素表现方式

## 用法

```tsx live
<Wrapper>
  <View display="block" s-100 bgBlue300>
    <View display="inline" s-50 bgBlue50></View>
  </View>
</Wrapper>
```

## 快捷属性

Styli 提供了大量快捷属性方便开发者

```tsx live
<Wrapper>
  <View block s-100 bgBlue300>
    <View inline bgBlue50>
      哈哈哈
    </View>
  </View>
</Wrapper>
```

> - block 为 `display: block`
> - inline 为 `display: inline`
> - inlineBlock 为 `display: inlineBlock`
> - hide 为 `display: none`
> - grid 为 `display: grid`
> - table 为 `display: table`

**注意: 对于 flex 布局，Styli 提供更为便捷的[布局引擎](layout-engine)，不必显示设置 display: flex.**

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View toCenter s-100 bgBlue300 relative>
    <View relative absolute--hover L-10 T-10 bgBlue50>
      hover
    </View>
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View toCenter s-100 display={['fixed', 'absolute', 'relative', 'static']} bgBlue300>
    media query
  </View>
</Wrapper>
```
