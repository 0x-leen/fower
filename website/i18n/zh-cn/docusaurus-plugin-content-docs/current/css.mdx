Styli 原子属性覆盖了常见的样式需求，但对于稍复杂的样式，我们同时提供了 ** css ** 属性来满足您的需求。

```jsx live
<View
  css={{
    a: {
      ':hover': {
        color: '#FF00FF',
      },
      ':visited': {
        color: '#00FF00',
      },
    },
  }}
>
  <a href="https://github.com/forsigner/styli" target="_blank">
    Styli
  </a>
</View>
```

上面的代码将被编译为

```css
.styli-1910396923 a:hover {
  color: #ff00ff;
}

.styli-1910396923 a:visited {
  color: #00ff00;
}
```

```html
<div class="styli-1910396923 ">
  <a href="https://github.com/forsigner/styli" target="_blank"> Styli </a>
</div>
```
