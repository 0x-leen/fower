import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic props for controlling how flex items wrap.

| Atomic Props     | CSS Properties     |
| ---------------- | ------------------ |
| flexWrap={value} | flex-wrap: {value} |

## Wrap

Use `flexWrap="wrap"` props to allow flex items to wrap.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box spaceX3 row flexWrap="wrap">
  <Box bgOrange400 m2 p4 w-300></Box>
  <Box bgOrange500 m2 p4 w-300></Box>
  <Box bgOrange400 m2 p4 w-300></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div spaceX3 row flexWrap="wrap">
  <div bgOrange400 m2 p4 w-300></div>
  <div bgOrange500 m2 p4 w-300></div>
  <div bgOrange400 m2 p4 w-300></div>
</div>
```

</TabItem>
</Tabs>

## No Wrap

Use `flexWrap="nowrap"` props to prevent flex items from wrapping, causing inflexible items to overflow the container if necessary.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box spaceX3 row flexWrap="nowrap">
  <Box bgOrange400 m2 p4 w-300></Box>
  <Box bgOrange500 m2 p4 w-300></Box>
  <Box bgOrange400 m2 p4 w-300></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div spaceX3 row flexWrap="nowrap">
  <div bgOrange400 m2 p4 w-300></div>
  <div bgOrange500 m2 p4 w-300></div>
  <div bgOrange400 m2 p4 w-300></div>
</div>
```

</TabItem>
</Tabs>
