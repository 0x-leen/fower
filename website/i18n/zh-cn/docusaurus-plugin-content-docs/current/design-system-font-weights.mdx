title: Font weights
---

Styli provides a set of preset font weights values.

```jsx
const config = {
  theme: {
    fontWeights: {
      hairline: 100,
      thin: 200,
      light: 300,
      normal: 400,
      medium: 500,
      semibold: 600,
      bold: 700,
      extrabold: 800,
      black: 900,
    },
  },
}
```
