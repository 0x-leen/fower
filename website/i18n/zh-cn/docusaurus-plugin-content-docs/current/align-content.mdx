id: align-content
title: Align Content
---

Atomic props for controlling how rows are positioned in multi-row flex and grid containers.

| Atomic Props         | CSS Properties          |
| -------------------- | ----------------------- |
| alignContent={value} | align-content: {value}; |

## Start

Use `alignContent="flex-start"` to pack rows in a container against the start of the cross axis.

```jsx live
<Box row bgAmber100 w-350 h-280 alignContent="flex-start">
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
</Box>
```

## Center

Use `alignContent="center"` to pack rows in a container in the center of the cross axis.

```jsx live
<Box row bgAmber100 w-350 h-320 alignContent="center">
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
</Box>
```

## End

Use `alignContent="flex-end"` to pack rows in a container against the end of the cross axis.

```jsx live
<Box row bgAmber100 w-350 h-320 alignContent="flex-end">
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
</Box>
```

## Space between

Use `alignContent="space-between"` to distribute rows in a container such that there is an equal amount of space between each line.

```jsx live
<Box row bgAmber100 w-350 h-320 alignContent="space-between">
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
</Box>
```

## Space around

Use `alignContent="space-around"` to distribute rows in a container such that there is an equal amount of space around each line.

```jsx live
<Box row bgAmber100 w-350 h-320 alignContent="space-evenly">
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
</Box>
```

## Space evenly

Use `alignContent="space-evenly"` to distribute rows in a container such that there is an equal amount of space around each item, but also accounting for the doubling of space you would normally see between each item when using content-around.

```jsx live
<Box row bgAmber100 w-350 h-320 alignContent="space-around">
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
  <Box m2 bgAmber400 square-100></Box>
</Box>
```
