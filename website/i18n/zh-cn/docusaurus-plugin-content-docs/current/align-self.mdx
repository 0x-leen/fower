import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic props for controlling how an individual flex or grid item is positioned along its container's cross axis.

| Atomic Props      | CSS Properties       |
| ----------------- | -------------------- |
| alignSelf={value} | align-self: {value}; |

## Start

Use `alignSelf="flex-start"` to align an item to the start of the container's cross axis, despite the container's `align-items` value.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box row spaceX4 alignItems="stretch">
  <Box bgOrange400 w-100 py6></Box>
  <Box bgBlue400 w-100 py4 alignSelf="flex-start"></Box>
  <Box bgOrange400 w-100 py8></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div row spaceX4 alignItems="stretch">
  <div bgOrange400 w-100 py6></div>
  <div bgBlue400 w-100 py4 alignSelf="flex-start"></div>
  <div bgOrange400 w-100 py8></div>
</div>
```

</TabItem>
</Tabs>
