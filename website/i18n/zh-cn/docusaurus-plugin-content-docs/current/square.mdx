import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

使用 `square-{value}` 格式的 Props 快速把元素变为正方形。

| Atomic Props                    | CSS Properties                   |
| ------------------------------- | -------------------------------- |
| square-{value} / square={value} | width: {value}; height: {value}; |

## Usage

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box toLeft white>
  <Box square-50 bgYellow400 toCenter>
    square-50
  </Box>
  <Box square-5rem bgGreen400 toCenter>
    square-5rem
  </Box>
  <Box square={100} bgBlue400 toCenter>
    square={'{100}'}
  </Box>
  <Box square="150px" bgPurple400 toCenter>
    square="150px"
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div toLeft white>
  <div square-50 bgYellow400 toCenter>square-50</div>
  <div square-5rem bgGreen400 toCenter>square-5rem</div>
  <div square="100" bgBlue400 toCenter>square="100"</div>
  <div square="150px" bgPurple400 toCenter>square="150px"</div>
</div>
```

</TabItem>
</Tabs>
