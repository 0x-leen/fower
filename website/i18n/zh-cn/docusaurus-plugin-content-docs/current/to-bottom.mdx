Atomic Props to make children elements align to bottom.

| Atomic Props | Effect                                 |
| ------------ | -------------------------------------- |
| toBottom     | Make children elements align to bottom |

## Usage

Use `toBottom` to make children elements align to bottom.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box toBottom bgGray100 square-200>
  <Box square-60 bgAmber400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div toBottom bgGray100 square-200>
  <div square-60 bgAmber400 rounded-8></div>
  <div square-80 bgBlue400 rounded-8></div>
</div>
```

</TabItem>
</Tabs>
