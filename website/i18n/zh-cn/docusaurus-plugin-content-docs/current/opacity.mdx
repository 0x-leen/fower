| Atomic Props                      | CSS Properties   |
| --------------------------------- | ---------------- |
| opacity-{value} / opacity={value} | opacity: {value} |

```jsx live
<View row toEvenly bgGreen50 p6 white>
  <View toCenter s-100 bgGreen500 opacity-20>
    opacity-20
  </View>
  <View toCenter s-100 bgGreen500 opacity-40>
    opacity-40
  </View>
  <View toCenter s-100 bgGreen500 opacity-60>
    opacity-60
  </View>
  <View toCenter s-100 bgGreen500 opacity={80}>
    opacity-80
  </View>
  <View toCenter s-100 bgGreen500 opacity-100>
    opacity-100
  </View>
</View>
```
