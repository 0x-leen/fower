Use atomic props to control the width of an element. width is a property used frequently, so we use abbreviations.

| Atomic Props          | CSS Properties  |
| --------------------- | --------------- |
| w-{value} / w={value} | width: {value}; |

## Fixed width

Use `w-{size}` Atomic Props to control the width of an element.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box white>
  <Box w-100 bgRed400 p2>
    w-100
  </Box>
  <Box w-10rem bgYellow400 p2>
    w-10rem
  </Box>
  <Box w={100} bgGreen400 p2>
    w={'{100}'}
  </Box>
  <Box w="400px" bgBlue400 p2>
    w="400px"
  </Box>
  <Box w="80%" bgPurple400 p2>
    w="80%"
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div white>
  <div w-100 bgRed400 p2>w-100</div>
  <div w-10rem bgYellow400 p2>w-10rem</div>
  <div w="100" bgGreen400 p2>w="100"</div>
  <div w="400px" bgBlue400 p2>w="400px"</div>
  <div w="80%" bgPurple400 p2>w="80%"</div>
</div>
```

</TabItem>
</Tabs>

## Fluid Width

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box white textCenter spaceY2>
  <Box toLeft>
    <Box w={1 / 2} bgOrange400 py-10>
      w={'{1 / 2}'}
    </Box>
    <Box w={1 / 2} bgRed400 py-10>
      w={'{1 / 2}'}
    </Box>
  </Box>

  <Box toLeft>
    <Box w={1 / 3} bgOrange400 py-10>
      w={'{1 / 3}'}
    </Box>
    <Box w={2 / 3} bgRed400 py-10>
      w={'{2 / 3}'}
    </Box>
  </Box>

  <Box toLeft>
    <Box w={1 / 4} bgOrange400 py-10>
      w={'{1 / 4}'}
    </Box>
    <Box w={3 / 4} bgRed400 py-10>
      w={'{3 / 4}'}
    </Box>
  </Box>

  <Box toLeft>
    <Box w={0.1} bgOrange400 py-10>
      w={'{0.1}'}
    </Box>
    <Box w={0.9} bgRed400 py-10>
      w={'{0.9}'}
    </Box>
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div white textCenter spaceY2>
  <div toLeft>
    <div w="1/2" bgOrange400 py-10>1/2</div>
    <div w="1/2" bgRed400 py-10>1/2</div>
  </div>

  <div toLeft>
    <div w="1/3" bgOrange400 py-10>1/3</div>
    <div w="2/3" bgRed400 py-10>2/3</div>
  </div>

  <div toLeft>
    <div w="1/4" bgOrange400 py-10>1/4</div>
    <div w="3/4" bgRed400 py-10>3/4</div>
  </div>

  <div toLeft>
    <div w="0.1" bgOrange400 py-10>0.1</div>
    <div w="0.9" bgRed400 py-10>0.9</div>
  </div>
</div>
```

</TabItem>
</Tabs>
