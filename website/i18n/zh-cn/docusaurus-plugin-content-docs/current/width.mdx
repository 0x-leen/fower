## 固定宽度

使用 `w-{size}` 格式的 Props 可以控制元素的宽度。

:::info
`width` 属性是高频使用的样式，所以简化为 `w`。
:::

```jsx live
<Wrapper>
  <View w-50 bgRed200>
    Hello
  </View>
  <View w-5rem bgYellow200>
    Hello
  </View>
  <View w={50} bgGreen200>
    Hello
  </View>
  <View w="100px" bgBlue200>
    Hello
  </View>
  <View w="20%" bgPurple200>
    Hello
  </View>
</Wrapper>
```

## 最小宽度

使用 `minW-{size}` 格式的 Props 可以控制元素的最小宽度。

```jsx live
<Wrapper>
  <View minW-100 bgBlue300>
    Hello
  </View>
</Wrapper>
```

## 最大宽度

使用 `maxW-{size}` 格式的 Props 可以控制元素的最大宽度。

```jsx live
<Wrapper>
  <View maxW-100 bgBlue300>
    Hello
  </View>
</Wrapper>
```

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View toCenter s-100 bgBlue300 w--hover={120}>
    hover
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View toCenter h-100 w={[100, 200, 300, 400]} bgBlue300>
    media query
  </View>
</Wrapper>
```
