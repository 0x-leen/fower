title: Font Size
---

Control the font size by `text-{value}` Atomic Props.

| Atomic Props                | CSS Properties      |
| --------------------------- | ------------------- |
| text-{value} / text={value} | font-size: {value}; |

## Usage

```jsx live
<Box>
  <Box textSM>Lorem ipsum dolor sit amet</Box>
  <Box text={20}>Lorem ipsum dolor sit amet</Box>
  <Box text-2rem>Lorem ipsum dolor sit amet</Box>
</Box>
```
