## Installation

```bash npm2yarn
npm install @fower/react
```

## Usage

```tsx
import { Box } from '@fower/react'
```

```jsx live
<Box bgGray100 p6>
  <Box toCenterY p-10 w-260 bgWhite rounded-10 shadowMD>
    <Box as="img" circle-48 src="/img/jobs.jpg" />
    <Box ml-10>
      <Box textXL fontBold>
        Steve Jobs
      </Box>
      <Box gray800>Co-founder of Apple Inc.</Box>
    </Box>
  </Box>
</Box>
```

More documentation is available at [use-with-react](/docs/use-with-react).
