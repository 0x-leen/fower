## Installation

```bash npm2yarn
npm install @fower/core
```

## Usage

When you customize an atomic props using `addAtom` or `composeAtom`,
you should extend AtomicProps types to get editor IntelliSense:

```ts
declare module '@fower/atomic-props' {
  export interface AtomicProps {
    textBody?: boolean

    brandLight?: boolean
    brandLighter?: boolean
    brandLightest?: boolean

    brandPrimary?: boolean

    brandDark?: boolean
    brandDarker?: boolean
    brandDarkest?: boolean

    bgBrandPrimary?: boolean

    borderBrandPrimary?: boolean

    formInput?: boolean
  }
}
```
