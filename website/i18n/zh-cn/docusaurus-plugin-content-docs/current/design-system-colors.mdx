</TabItem>

## Customize theme color

You can customize theme color with `setTheme`:

```jsx
setTheme({
  colors: {
    brandLight: '#e0f2fe',
    brandLighter: '#bae6fd',
    brandLightest: '#7dd3fc',
    brandPrimary: '#38bdf8',
    brandDark: '#0ea5e9',
    brandDarker: '#0284c7',
    brandDarkest: '#0369a1',
  },
})
```

Then you can use it with text color, background color, border color:

<Tabs
  defaultValue="React"
  values={[
    { label: 'React', value: 'React', },
    { label: 'Vue', value: 'Vue', },
  ]
}>
<TabItem value="React">

```jsx live
<Box toEvenly>
  <Box toCenter p4 brandPrimary>
    Text color
  </Box>
  <Box toCenter p4 bgBrandLighter brandDarkest>
    Background color
  </Box>
  <Box toCenter p4 border-2 borderBrandDark>
    Border color
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div toEvenly>
  <div toCenter p4 brandPrimary>Text color</div>
  <div toCenter p4 bgBrandLighter brandDarkest>Background color</div>
  <div toCenter p4 border-2 borderBrandDark>Border color</div>
</div>
```

</TabItem>
</Tabs>
