Fower comes with a predefined set of commonly used breakpoints.

```jsx
const config = {
  theme: {
    breakpoints: {
      sm: '640px',
      md: '768px',
      lg: '1024px',
      xl: '1280px',
      '2xl': '1536px',
    },
  },
}
```

## Customize Breakpoints

You can customize Breakpoints with `setTheme`:

```jsx
setTheme({
  breakpoints: {
    '3xl': '1600px',
  },
})
```

Then you can use it like this:

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box bgGreen400 square12 square20--sm square32--md square48--3xl></Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div bgGreen400 square12 square20--sm square32--md square48--3xl></div>
```

</TabItem>
</Tabs>
