很多 CSS 框架，都会声称自己是 Mobile-first，是响应式的。比如两个非常知名的 CSS 框架，Bootstrap、Bulma，是这样描述自己的:

> Quickly design and customize responsive mobile-first sites with Bootstrap.

> Bulma is a mobile-first framework.

这是传统 CSS 框架的响应式，大部分都是只 Grid System 的响应式，这是狭义的响应式。真正的响应式应该每个样式都能响应式的的，而不是只是支持 Grid System 响应式。

Fower 的每个原子属性都支持响应式，而不是只是某个容器支持响应式，这可以让你更轻松地实现复杂的响应式界面。

## Postfix

| Breakpoint postfix | Minimum width | CSS                                |
| ------------------ | ------------- | ---------------------------------- |
| sm                 | 640px         | @media (min-width: 640px) { ... }  |
| md                 | 768px         | @media (min-width: 768px) { ... }  |
| lg                 | 1024px        | @media (min-width: 1024px) { ... } |
| xl                 | 1280px        | @media (min-width: 1280px) { ... } |
| 2xl                | 1536px        | @media (min-width: 1536px) { ... } |

## Usage

Use `--{postfix}` like `--sm`, `--md`, `--lg` to set responsive style.

```jsx live
<Box textSM textLG--sm text4XL--md text6XL--lg>
  Lorem ipsum dolor sit amet
</Box>
```
