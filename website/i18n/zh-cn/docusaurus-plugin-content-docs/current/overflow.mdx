title: overflow 溢出
---

使用`overflow{X|Y}` 系列属性可以快速元素溢出行为

## 溢出

```jsx live
<Wrapper>
  <View s-100 overflow="scroll" bgBlue300 p-10>
    <View s-120 bgRed300></View>
  </View>
</Wrapper>
```

## 横轴溢出

```jsx live
<Wrapper>
  <View s-100 overflowY="hidden" overflowX="scroll" bgBlue300 p-10>
    <View s-120 bgRed300></View>
  </View>
</Wrapper>
```

## 纵轴溢出

```jsx live
<Wrapper>
  <View s-100 overflowX="hidden" overflowY="scroll" bgBlue300 p-10>
    <View s-120 bgRed300></View>
  </View>
</Wrapper>
```

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View s-100 oyHidden--hover oxScroll bgBlue300 p-10>
    <View s-120 bgRed300></View>
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View toCenter s-100 overflow={['hidden', 'scroll', 'auto', 'visible']} bgBlue300>
    media query
  </View>
</Wrapper>
```
