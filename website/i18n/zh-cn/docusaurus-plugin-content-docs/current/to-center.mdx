title: To Center
---

让子元素全部向中心对齐。

| Atomic Props | Effect                                 |
| ------------ | -------------------------------------- |
| toCenter     | Make children elements align to center |

## 用法

### toCenter with row

```jsx live
<Box toCenter bgGray100 s-200>
  <Box s-60 bgAmber400 rounded-8></Box>
  <Box s-80 bgBlue400 rounded-8></Box>
</Box>
```

### toCenter with column

```jsx live
<Box column toCenter bgGray100 s-200>
  <Box s-60 bgAmber400 rounded-8></Box>
  <Box s-80 bgBlue400 rounded-8></Box>
</Box>
```
