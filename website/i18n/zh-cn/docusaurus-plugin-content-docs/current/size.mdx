使用 `s-{value}` 格式的 Props 快速把元素变为正方形。

| Atomic Props          | CSS Properties                   |
| --------------------- | -------------------------------- |
| s-{value} / s={value} | width: {value}; height: {value}; |

## Usage

```jsx live
<Box toLeft white>
  <Box s-50 bgYellow400 toCenter>
    s-50
  </Box>
  <Box s-5rem bgGreen400 toCenter>
    s-5rem
  </Box>
  <Box s={100} bgBlue400 toCenter>
    s={'{100}'}
  </Box>
  <Box s="150px" bgPurple400 toCenter>
    s="150px"
  </Box>
</Box>
```
