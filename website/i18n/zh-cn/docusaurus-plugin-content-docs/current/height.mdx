title: Height
---

使用 h-{value} 格式的 Props 可以控制元素的高度。

| Atomic Props          | CSS Properties   |
| --------------------- | ---------------- |
| h-{value} / h={value} | height: {value}; |

## Usage

```jsx live
<Box bgGray200 white square-200>
  <Box maxH-100 h-200 bgRed400 toCenter>
    maxH-100
  </Box>
</Box>
```
