Atomic Props to make children elements align to right.

import { Layout4 } from '@site/src/components/Layout'

| Atomic Props | Effect                                |
| ------------ | ------------------------------------- |
| toRight      | Make children elements align to right |

## Usage

Use `toTop` to make children elements align to right.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box toRight bgGray100 square-200>
  <Box square-60 bgAmber400 rounded-8></Box>
  <Box square-80 bgBlue400 rounded-8></Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div toRight bgGray100 square-200>
  <div square-60 bgAmber400 rounded-8></div>
  <div square-80 bgBlue400 rounded-8></div>
</div>
```

</TabItem>
</Tabs>
