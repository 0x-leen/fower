import Tabs from '@theme/Tabs'
import TabItem from '@theme/TabItem'

Atomic Props for controlling how flex items grow.

| Atomic Props     | CSS Properties      |
| ---------------- | ------------------- |
| flexGrow={value} | flex-grow: {value}; |

## Grow

Use `flexGrow={1}` props to allow a flex item to grow to fill any available space.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box spaceX3 row>
  <Box bgOrange400 p4>
    Normal
  </Box>
  <Box bgOrange500 p4 flexGrow={1}>
    Grow
  </Box>
  <Box bgOrange400 p4>
    Normal
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div spaceX3 row>
  <div bgOrange400 p4>Normal</div>
  <div bgOrange500 p4 flexGrow="1">Grow</div>
  <div bgOrange400 p4>Normal</div>
</div>
```

</TabItem>
</Tabs>

## Don't grow

Use `flexGrow={0}` to prevent a flex item from growing.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box spaceX3 row>
  <Box bgOrange400 p4 flexGrow={1}>
    Grow
  </Box>
  <Box bgOrange500 p4 flexGrow={0}>
    Not Grow
  </Box>
  <Box bgOrange400 p4 flexGrow={1}>
    Grow
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div spaceX3 row>
  <div bgOrange400 p4 flexGrow="1">Grow</div>
  <div bgOrange500 p4 flexGrow="0">Not Grow</div>
  <div bgOrange400 p4 flexGrow="1">Grow</div>
</div>
```

</TabItem>
</Tabs>
