title: Flex Grow
---

Atomic Props for controlling how flex items grow.

| Atomic Props     | CSS Properties      |
| ---------------- | ------------------- |
| flexGrow={value} | flex-grow: {value}; |

## Grow

Use `flexGrow={1}` props to allow a flex item to grow to fill any available space.

```jsx live
<Box spaceX3 row>
  <Box bgAmber400 p4>
    Normal
  </Box>
  <Box bgAmber500 p4 flexGrow={1}>
    Grow
  </Box>
  <Box bgAmber400 p4>
    Normal
  </Box>
</Box>
```

## Don't grow

Use `flexGrow={0}` to prevent a flex item from growing.

```jsx live
<Box spaceX3 row>
  <Box bgAmber400 p4 flexGrow={1}>
    Grow
  </Box>
  <Box bgAmber500 p4 flexGrow={0}>
    Not Grow
  </Box>
  <Box bgAmber400 p4 flexGrow={1}>
    Grow
  </Box>
</Box>
```
