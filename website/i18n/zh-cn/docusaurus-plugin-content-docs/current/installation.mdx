Styli 是支持多个框架，包括 React、Vue、React Native、Taro 等。

<Tabs
  defaultValue="react"
  values={[
    { label: 'React', value: 'react' },
    { label: 'Vue', value: 'vue' },
    { label: 'React Native', value: 'react-native' },
    { label: 'Taro', value: 'taro' },
  ]}
>
  <TabItem value="react">

```bash
npm i @styli/react
```

  </TabItem>

  <TabItem value="vue">

```bash
npm i @styli/vue
```

  </TabItem>

  <TabItem value="react-native">

```bash
npm i @styli/react-native
```

  </TabItem>

  <TabItem value="taro">

```bash
npm i @styli/taro
```

  </TabItem>
</Tabs>
