id: min-height
title: Min Height
---

使用 minH-{value} 格式的 Props 可以控制元素的最小高度。

| Atomic Props                | CSS Properties       |
| --------------------------- | -------------------- |
| minH-{value} / minH={value} | min-height: {value}; |

## Usage

```jsx live
<Box toCenter white h-100 bgGray100>
  <Box minH-50 bgRed400 p2>
    minH-50
  </Box>
  <Box minH-5rem bgYellow400 p2>
    minH-5rem
  </Box>
  <Box minH={50} bgGreen400 p2>
    minH={'{50}'}
  </Box>
  <Box minH="100px" bgBlue400 p2>
    minH="100px"
  </Box>
  <Box minH="80%" bgPurple400 p2>
    minH="80%"
  </Box>
</Box>
```
