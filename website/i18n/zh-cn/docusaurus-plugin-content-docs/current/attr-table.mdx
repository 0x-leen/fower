## 常规

### Size

| 属性                    | 用法               | 表示     |
| ----------------------- | ------------------ | -------- |
| [w](width)              | `<View w-10 />`    | 宽度     |
| [h](height)             | `<View h-10 />`    | 高度     |
| [maxW](width#最大宽度)  | `<View maxW-10 />` | 最大宽度 |
| [minW](width#最小宽度)  | `<View minW-10 />` | 最小宽度 |
| [maxH](height#最大高度) | `<View maxH-10 />` | 最大高度 |
| [minH](height#最小高度) | `<View minH-10 />` | 最小高度 |

### Shape

| 属性               | 用法                 | 表示   |
| ------------------ | -------------------- | ------ |
| [s](shape#正方形)  | `<View s-10 />`      | 正方形 |
| [s](shape#长方形)  | `<View s-10-20 />`   | 长方形 |
| [circle](shape#圆) | `<View circle-10 />` | 圆     |

### Font

| 属性                                                      | 用法                      | 表示         |
| --------------------------------------------------------- | ------------------------- | ------------ |
| [f](typography#字体大小)                                  | `<View f-10 />`           | 字体大小     |
| [c](typography#文字颜色)                                  | `<View color="red20" />`  | 字体颜色     |
| [lh](typography#行高)                                     | `<View lh-10 />`          | 行高         |
| [fontWeight](typography#字体粗细)                         | `<View fontWeight-100 />` | 字体粗细     |
| [text{Center\|Left\|Right\|Justify}](typography#文本对齐) | `<View textCenter />`     | 文本对齐方式 |

### Background

| 属性                       | 用法                            | 表示         |
| -------------------------- | ------------------------------- | ------------ |
| [bg](background)           | `<View bg="red20" />`           | 背景         |
| [bgColor](background#颜色) | `<View bgColor="red20" />`      | 背景颜色     |
| [bgImg](background#图片)   | `<View bgImg="https://..." />`  | 背景图片     |
| [bgPos](background#位置)   | `<View bgPos="right bottom" />` | 背景图片位置 |
| [bgSize](background#大小)  | `<View bgSize="100% 100%" />`   | 背景图片大小 |

### Padding

| 属性                          | 用法                   | 表示                   |
| ----------------------------- | ---------------------- | ---------------------- |
| [p](padding)                  | `<View p-10 />`        | 四周内边距             |
| [p{t\|r\|b\|l}](padding#一边) | `<View pt-10 pr-10 />` | 一边的内边距           |
| [p{x\|y}](padding#水平)       | `<View px-10 py-10 />` | 水平或垂直方向的内边距 |
| [p](padding#水平和垂直)       | `<View p-10-20 />`     | 水平和垂直方向的内边距 |

### Margin

| 属性                         | 用法                   | 表示                   |
| ---------------------------- | ---------------------- | ---------------------- |
| [m](margin)                  | `<View m-10 />`        | 四周外边距             |
| [m{t\|r\|b\|l}](margin#一边) | `<View mt-10 mr-10 />` | 一边的外边距           |
| [m{x\|y}](margin#水平)       | `<View mx-10 my-10 />` | 水平或垂直方向的外边距 |
| [m](margin#水平和垂直)       | `<View m-10-20 />`     | 水平和垂直方向的外边距 |
| [m](margin#负-margin)        | `<View m--10 />`       | 负值 margin            |

### Position
