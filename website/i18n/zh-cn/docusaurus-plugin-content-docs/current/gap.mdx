title: Gap

| Atomic Props      | CSS Properties       |
| ----------------- | -------------------- |
| gap-{value}       | gap: {value};        |
| rowGap-{value}    | row-gap: {value};    |
| columnGap-{value} | column-gap: {value}; |

## Gap

```jsx live
<Box grid gridTemplateColumns-2 gap-20 white fontBold textXL>
  <Box py1 rounded-4 toCenter bgTeal400>
    1
  </Box>
  <Box py1 rounded-4 toCenter bgTeal400>
    2
  </Box>
  <Box py1 rounded-4 toCenter bgTeal400>
    3
  </Box>
  <Box py1 rounded-4 toCenter bgTeal400>
    4
  </Box>
</Box>
```
