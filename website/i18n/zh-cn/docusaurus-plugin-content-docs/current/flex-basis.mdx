---

Flexbox 是一种用于按行或按列布局元素的一维布局方法 。元素可以膨胀以填充额外的空间, 收缩以适应更小的空间。Styli 支持大部分常用的 flexbox 属性，可以满足您大部分布局场景。

如果你不熟悉弹性盒子，建议先通过 MDN [了解 flexbox](https://developer.mozilla.org/zh-CN/docs/Learn/CSS/CSS_layout/Flexbox)。

## Flex Direction

使用 `row` 和 `column` 控制 flex 布局的方向，为了你的 UI 结构更加简洁，当你设置了 `row` 和 `column`，你不需要再设置 `display: flex` 样式属性，Styli 会自动帮你添加该属性。

```jsx live
<View row>
  <View s-100 bgBlue300></View>
  <View s-100 bgGreen300></View>
</View>
```
