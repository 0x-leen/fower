Atomic props for controlling how an element is positioned.

| Atomic Props     | CSS Properties      |
| ---------------- | ------------------- |
| static           | position: static;   |
| fixed            | position: fixed;    |
| absolute         | position: absolute; |
| relative         | position: relative; |
| sticky           | position: sticky;   |
| position={value} | position: {value};  |

## Static

Use `static` props to set static position for an element.

```jsx live
<Box relative s-160 bgBlue100>
  <Box bgAmber400 s-100 static toCenter>
    static
  </Box>
</Box>
```

## Relative

Use `relative` props to set relative position for an element.

```jsx live
<Box relative s-160 bgBlue100>
  relative
  <Box bgBlue400-T20 s-80 absolute bottom-0 toCenter>
    absolute
  </Box>
</Box>
```

## Absolute

Use `absolute` props to set absolute position for an element.

```jsx live
<Box relative s-160 bgBlue100>
  <Box bgBlue400-T20 s-80 absolute bottom-0 toCenter>
    absolute
  </Box>
</Box>
```

## Fixed

Use `fixed` props to set fixed position for an element.

```jsx live
<Box toCenter bgWhite shadow rounded-8 s-80 fixed bottom-10>
  fixed
</Box>
```

## Sticky

Use `sticky` props to set sticky position for an element.

```jsx live
<Box relative s-200 overflow="auto" bgGray100 p4>
  <Box>Lorem ipsum dolor sit amet, consectetur adipisicing elit.</Box>
  <Box toCenter bgYellow200 shadow rounded-8 s-80 sticky top-0>
    sticky
  </Box>
  <Box>
    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Assumenda, quia temporibus
    eveniet a libero incidunt suscipit laborum, rerum accusantium modi quidem, ipsam illum quis sed
    voluptatum quae eum fugit earum.
  </Box>
</Box>
```
