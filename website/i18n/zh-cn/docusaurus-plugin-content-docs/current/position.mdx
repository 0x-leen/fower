Atomic props for controlling how an element is positioned.

| Atomic Props     | CSS Properties      |
| ---------------- | ------------------- |
| static           | position: static;   |
| fixed            | position: fixed;    |
| absolute         | position: absolute; |
| relative         | position: relative; |
| sticky           | position: sticky;   |
| position={value} | position: {value};  |

## Static

Use `static` props to set static position for an element.

<Tabs
defaultValue="React"
values={[
{ label: 'React', value: 'React', },
{ label: 'Vue', value: 'Vue', },
]
}>
<TabItem value="React">

```jsx live
<Box relative square-160 bgBlue100>
  <Box bgAmber400 square-100 static toCenter>
    static
  </Box>
</Box>
```

</TabItem>
<TabItem value="Vue">

```html
<div relative square-160 bgBlue100>
  <div bgAmber400 square-100 static toCenter>static</div>
</div>
```

</TabItem>
</Tabs>
