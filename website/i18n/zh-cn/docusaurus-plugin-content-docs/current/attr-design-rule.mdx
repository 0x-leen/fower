title: 属性设计规范
---

为了减少使用成本，Styli 支持大量简写属性进行快捷开发。

```tsx live
<Wrapper>
  <View s-100 bgBlue200 toCenter style={{ color: 'red' }}>
    Hi, Styli
  </View>
  <View s-100 bgBlue200 toCenter color="red">
    Hi, Styli
  </View>
  <View s-100 bgBlue200 toCenter color="red">
    Hi, Styli
  </View>
  <View s-100 bgBlue200 toCenter red>
    Hi, Styli
  </View>
</Wrapper>
```

## 属性值作为属性

我们将多个常用的可枚举属性值的属性进行了筛选。提取整合出了以下几个常用类别

### 直接将属性值作为属性

| 属性                                     | 实例                 | 表示      |
| ---------------------------------------- | -------------------- | --------- |
| [display](display)                       | `<View inline />`    | 元素显示  |
| [position](position)                     | `<View absolute />`  | 元素定位  |
| [flexDirection](flex-box#flex-direction) | `<View row />`       | flex 方向 |
| [flexWrap](flex-box#flex-wrap)           | `<View wrap />`      | flex 换行 |
| textTransform                            | `<View uppercase />` | 文本转换  |
| [color](typography#文字颜色)             | `<View red />`       | 文字颜色  |

### 属性值添加前缀/后缀作为属性

当属性值与其他属性值或 DOM 属性冲突时，则添加属性前缀/后缀

| 属性                                           | 实例                    | 表示       |
| ---------------------------------------------- | ----------------------- | ---------- |
| [cursor](cursor)                               | `<View cursorAuto />`   | 指针       |
| [justifyContent](flex-box#主轴-justify-conten) | `<View justifyStart />` | 主轴排序   |
| [alignItems](flex-box#交叉轴-align-items)      | `<View itemsStart />`   | 交叉轴排序 |
| [outline](outline)                             | `<View outlineNone />`  | 轮廓       |
| [textAlign](typography#文本对齐)               | `<View textCenter />`   | 文本对齐   |

## 单字符属性

单字符属性是最常用的属性的简写。单字符属性通常以原属性首字母作为新属性。

| 简写                        | 属性                  | 表示     |
| --------------------------- | --------------------- | -------- |
| [f](typography#字体大小)    | fontSize              | 字体大小 |
| [c](typography#文字颜色)    | color                 | 字体颜色 |
| [w](width)                  | width                 | 宽度     |
| [h](height)                 | height                | 高度     |
| [p](padding)                | padding               | 内边距   |
| [m](margin)                 | margin                | 外边距   |
| [T\|R\|B\|L](position#定位) | top/right/bottom/left | 定位     |

## 常规属性

| 简写                                           | 属性                     | 表示     |
| ---------------------------------------------- | ------------------------ | -------- |
| [border{T\|R\|B\|L}](rounded)                  | `<View borderT="1px" />` | 元素边框 |
| [rounded{T\|R\|B\|L\|TL\|TR\|BL\|BR}](rounded) | `<View rounded-20 />`    | 元素圆角 |
| [opacity](opacity)                             | `<View opacity-20 />`    | 透明度   |
| [shadow](shadow)                               | `<View shadow="1px" />`  | 阴影     |

## 功能属性

Styli 中集成整合了包含多个属性的复合属性。

| 属性                                         | 实例                                         | 表示                              |
| -------------------------------------------- | -------------------------------------------- | --------------------------------- |
| [s](shape)                                   | `<View s-100 />`                             | 矩形                              |
| [space{X\|Y}](space)                         | `<View space-100 />`                         | 子元素间隔                        |
| [ellipsis](ellipsis)                         | `<View ellipsis2-100 />`                     | 文本溢出省略                      |
| [debug{All\|Children}](debug)                | `<View debug />`                             | 自动为元素添加边框                |
| [reset](reset)                               | `<View reset />`                             | 重置浏览器默认样式                |
| [styliIgnore](styli-ignore)                  | `<Button styliIgnore={['center']} center />` | 让 styli 忽略对属性的处理         |
| [styliName](styli-name)                      | `<View styliName="user-card" />`             | 组件语义化                        |
