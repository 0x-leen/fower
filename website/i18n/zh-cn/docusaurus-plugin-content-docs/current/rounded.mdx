id: rounded
title: rounded 圆角
---

使用 `rounded{T|R|B|L|TL|TR|BL|BR}-{size}` 格式的 Props 可以控制元素的圆角大小。

## 四周圆角

```jsx live
<Wrapper>
  <View s-100 bgYellow200 rounded-6></View>
  <View s-100 bgRed200 rounded-10px></View>
  <View s-100 bgGreen200 rounded={20}></View>
  <View s-100 bgBlue200 rounded="50px"></View>
</Wrapper>
```

## 某一方向

```jsx live
<Wrapper>
  <View s-100 bgYellow200 roundedT-20></View>
  <View s-100 bgRed200 roundedR-20></View>
  <View s-100 bgGreen200 roundedB-20></View>
  <View s-100 bgBlue200 roundedL-20></View>
</Wrapper>
```

## 单个圆角

```jsx live
<Wrapper>
  <View s-100 bgYellow200 roundedTL-20></View>
  <View s-100 bgRed200 roundedTR-20></View>
  <View s-100 bgGreen200 roundedBL-20></View>
  <View s-100 bgBlue200 roundedBR-20></View>
</Wrapper>
```

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View s-100 bgYellow200 rounded-20--hover></View>
  <View s-100 bgGreen200 rounded-20--active></View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View toCenter s-200 rounded={[30, 50, 70, 90]} bgBlue300>
    rounded = [30, 50, 70, 90]
  </View>
</Wrapper>
```
