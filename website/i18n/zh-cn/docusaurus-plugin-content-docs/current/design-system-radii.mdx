Styli provides a set of preset border radius values.

```jsx
const config = {
  theme: {
    radii: {
      none: 0,
      sm: 2,
      base: 4,
      md: 6,
      lg: 8,
      xl: 12,
      '2xl': 16,
      '3xl': 24,
      full: 9999,
    },
  },
}
```

## Customize Border radius

You can customize Border radius with `setTheme`:

```jsx
setTheme({
  radii: {
    '4xl': 28,
  },
})
```

Then you can use it as `rounded4XL`:

<Tabs
  defaultValue="React"
  values={[
    { label: 'React', value: 'React', },
    { label: 'Vue', value: 'Vue', },
  ]
}>
<TabItem value="React">

```jsx live
<Box bgGreen400 square32 rounded4XL></Box>
```

</TabItem>
<TabItem value="Vue">

```html
<html bgGreen400 square32 rounded4XL></html>
```

</TabItem>
</Tabs>
