---

使用 `cursor` 可快速设置光标表现方式

## 用法

```tsx live
<Wrapper>
  <View cursor="pointer" s-100 bgBlue300></View>
</Wrapper>
```

## 快捷属性

Styli 提供了大量快捷属性方便开发者

```tsx live
<Wrapper>
  <View cursorPointer s-100 bgBlue300></View>
  <View cursorProgress s-100 bgBlue300></View>
  <View cursorCell s-100 bgBlue300></View>
</Wrapper>
```
