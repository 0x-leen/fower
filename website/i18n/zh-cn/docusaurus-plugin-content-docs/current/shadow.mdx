title: shadow 阴影
---

使用 `shadow` 属性可快速设置阴影

## 用法

```jsx live
<Wrapper>
  <View center s-100 bgWhite shadow="0 24px 48px rgba(0, 0, 0, 0.2), 0 0 2px rgba(0,0,0,0.02)">
    shadow
  </View>
</Wrapper>
```

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View
    center
    s-100
    bgWhite
    shadow--hover="0 0 0 3px rgba(66, 153, 225, 0.5)"
    shadow="0 8px 16px rgba(0, 0, 0, 0.12), 0 0 2px rgba(0,0,0,0.02)"
  >
    hover
  </View>
  <View
    center
    s-100
    bgWhite
    shadow---active="0 0 0 3px rgba(66, 153, 225, 0.5)"
    shadow="0 8px 16px rgba(0, 0, 0, 0.12), 0 0 2px rgba(0,0,0,0.02)"
  >
    active
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View
    h-100
    w-200
    center
    shadow={[
      '0 1px 1px rgba(0, 0, 0, 0.12), 0 0 1px rgba(0,0,0,0.01)',
      '0 0 0 3px rgba(66, 153, 225, 0.5)',
      '0 24px 48px rgba(0, 0, 0, 0.2), 0 0 2px rgba(0,0,0,0.02)',
      '0 8px 16px rgba(0, 0, 0, 0.12), 0 0 2px rgba(0,0,0,0.02)',
    ]}
    bgWhite
  >
    media query
  </View>
</Wrapper>
```

## 设计系统

使用设计系统属性，让您的应用更有规范。

`shadow{SIZE}` 格式的 Props 可以控制元素的 shadow。

```jsx live
<Wrapper>
  <View center s-50 bgWhite shadowXS>
    XS
  </View>
  <View center s-60 bgWhite shadowSM>
    SM
  </View>
  <View center s-70 bgWhite bgWhite shadowBase>
    Base
  </View>
  <View center s-80 bgWhite shadowMD>
    MD
  </View>
  <View center s-90 bgWhite shadowLG>
    LG
  </View>
  <View center s-100 bgWhite shadowXL>
    XL
  </View>
  <View center s-110 bgWhite shadowXXL>
    XXL
  </View>
</Wrapper>
```
