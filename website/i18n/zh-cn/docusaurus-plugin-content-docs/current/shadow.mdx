title: Shadow
---

Use `shadow{SIZE}` Atomic Props controll element's shadow.

| Atomic Props   | CSS Properties      |
| -------------- | ------------------- |
| shadowSM       | box-shadow: {SM}    |
| shadow         | box-shadow: {Base}  |
| shadowMD       | box-shadow: {MD}    |
| shadowLG       | box-shadow: {LG}    |
| shadowXL       | box-shadow: {XL}    |
| shadow2XL      | box-shadow: {2XL}   |
| shadow={value} | box-shadow: {value} |

## Usage

```jsx live
<Box bgGray100 p-20 toCenterX toBetween>
  <Box toCenter square-60 bgWhite shadowSM>
    SM
  </Box>
  <Box toCenter square-80 bgWhite bgWhite shadow>
    Base
  </Box>
  <Box toCenter square-80 bgWhite shadowMD>
    MD
  </Box>
  <Box toCenter square-90 bgWhite shadowLG>
    LG
  </Box>
  <Box toCenter square-100 bgWhite shadowXL>
    XL
  </Box>
  <Box toCenter square-110 bgWhite shadow2XL>
    2XL
  </Box>
</Box>
```

## Default shadow

```jsx live
<Box bgGray100 p-20 toCenter>
  <Box toCenter square-100 bgWhite shadow>
    default
  </Box>
</Box>
```
