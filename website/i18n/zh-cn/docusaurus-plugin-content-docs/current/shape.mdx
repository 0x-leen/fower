## 正方形

使用 `s-{size}` 格式的 Props 快速设置元素为正方形。

```jsx live
<Wrapper>
  <View s-50 bgYellow200>
    Hello
  </View>
  <View s-5rem bgGreen200>
    Hello
  </View>
  <View s={100} bgBlue200>
    Hello
  </View>
  <View s="150px" bgPurple200>
    Hello
  </View>
</Wrapper>
```

## 长方形

使用 `s-{width}-{height}` 格式的 Props 快速设置元素为长方形。

```jsx live
<Wrapper>
  <View s-50-50 bgYellow200>
    Hello
  </View>
  <View s-50-5rem bgGreen200>
    Hello
  </View>
  <View s-5rem-5rem bgBlue200>
    Hello
  </View>
  <View s-50-150 bgPurple200>
    Hello
  </View>
</Wrapper>
```

## 圆

使用 `circle-{size}` 格式的 Props 快速设置元素为圆。

```jsx live
<Wrapper>
  <View circle-50 bgYellow200 toCenter>
    Hello
  </View>
  <View circle-5rem bgGreen200 toCenter>
    Hello
  </View>
  <View circle={100} bgBlue200 toCenter>
    Hello
  </View>
  <View circle="150px" bgPurple200 toCenter>
    Hello
  </View>
</Wrapper>
```

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View toCenter s-100 bgBlue300 s--hover={120}>
    hover
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View toCenter circle={[30, 50, 70, 90]} bgBlue300>
    media query
  </View>
</Wrapper>
```

## 设计系统

使用设计系统属性，让您的应用更有规范

```jsx live

    s1

    s2
```
