## develop-guide

### Install yarn

First step, install yarn:

https://yarn.bootcss.com/docs/install/#windows-stable

### Install deps

```bash
# In Styli project root
yarn install # Equal: lerna bootstrap --npm-client yarn --use-workspaces
```

### Build All

```bash
yarn build
```

### Run Example

```bash
cd packages/example
yarn start
```
