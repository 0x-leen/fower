## 使用

Styli 是跨框架的，下面我们看看在 React 的用法，使用 Styli 提供的 View、Text、Image 等组件，你可以快速实现复杂界面。

```jsx
import React from 'react'
import { View, Text, Image } from '@styli/react'

export default () => {
  return (
    <View centerY p-10 w-260 bgWhite rounded-10>
      <Image circle-48 src="/img/jobs.jpg" />
      <View ml-10>
        <View f-20 fontBold>
          Steve Jobs
        </View>
        <Text gray80>Co-founder of Apple Inc.</Text>
      </View>
    </View>
  )
}
```

Styli 巧妙地利用了 JSX 的 boolean(true) 类型 props 可以省略值的特性，实现了 "Key as Value" 模式，让 Styli 在是构建界面时更加简洁，同时由于使用了 tsx，我们得到了强类型的 "CSS in JS"。
