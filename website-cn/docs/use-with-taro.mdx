[Taro](https://taro-docs.jd.com/taro/docs/README/index.html) 是一个小程序框架，在 Taro 中使用 Styli 和在 React 中使用非常类似。

:::note
小程序不能动态添加 className, 因而 styli 只能将 Atomic Props 转换为内联 CSS
:::

## 安装

```bash
npm i @styli/taro -S
```

## 使用

从 **@styli/taro** 包中导入组件

```tsx
import { View, Image, Text } from '@styli/taro'
```

使用组件，实现一个用户信息卡片界面。

```jsx live
<View centerY p-10 w-260 bgWhite rounded-10 shadow>
  <Image circle-48 src="/img/jobs.jpg" />
  <View ml-10>
    <View f-20 fontBold>
      Steve Jobs
    </View>
    <Text gray80>Co-founder of Apple Inc.</Text>
  </View>
</View>
```
