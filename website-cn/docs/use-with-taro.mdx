[Taro](https://taro-docs.jd.com/taro/docs/README/index.html) 是一个 React-like 的框架，在 Taro 中使用 Styli 和在 React 中使用非常类似。

:::info
styli 是运行时框架，在 Taro 中只能将 Atomic CSS 转换为内联CSS，因而部分属性不可使用
:::

## 安装

```bash
npm i @styli/taro -S
```

## 使用

从 **@styli/taro** 包中导入组件

```tsx
import { View, Image, Text } from '@styli/taro'
```

使用组件，实现一个用户信息卡片界面。

```jsx live
<View centerY p-10 w-260 bgWhite rounded-10 shadow>
  <Image circle-48 src="/img/jobs.jpg" />
  <View ml-10>
    <View f-20 fontBold>
      Steve Jobs
    </View>
    <Text gray80>Co-founder of Apple Inc.</Text>
  </View>
</View>
```

## 配置

您可以查看 [配置](config)，[插件](plugin)和[主题](theme) 章节进行查阅

### 自定义组件

**@styli/taro** 中包装的 **taro component** 可能较旧，这时，您可以使用 [styled](styled) API 对 taro component 进行封装

```tsx
import { styled } from '@styli/taro'
import TaroComponent from 'tarojs/components'

export const View = styled(TaroComponent.View)
```
