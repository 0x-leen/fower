## Font Size

使用 `f-{size}` 格式的 Props 可以控制字体的大小。

:::note
`fontSize` 属性是高频使用的样式，所以简化为 `f`。
:::

```jsx live
<View>
  <Text f-20>王勃</Text>
  <Text f={40}> 李白</Text>
  <Text f-4rem> 杜甫</Text>
</View>
```

## Font Weight

有两种方式可以控制字体粗细，一种是使用预设值，另外一种是使用精确的传值。

使用预设值:

```jsx live
<View f-24>
  <View fontHairline>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontThin>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontLight>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontNormal>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontMedium>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontSemibold>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontBold>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontExtrabold>落霞与孤鹜齐飞，秋水共长天一色。</View>
</View>
```

使用精确传值:

```jsx live
<View f-24>
  <View fontWeight-100>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontWeight-400>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View fontWeight={400}>落霞与孤鹜齐飞，秋水共长天一色。</View>
</View>
```

## Line Height

有两种方式可以控制文本的行高，一种是使用预设值，另外一种是使用精确的传值。

使用预设值 `lhNone`、`lhTight`、`lhSnug`、`lhNormal`、`lhRelaxed`、`lhLoose`:

```jsx live
<View>
  <View lhNone>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lhTight>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lhSnug>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lhNormal>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lhRelaxed>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lhLoose>落霞与孤鹜齐飞，秋水共长天一色。</View>
</View>
```

使用精确传值:

```jsx live
<View f-20>
  <View lh-20>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lh-3em>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View lh="1.5em">落霞与孤鹜齐飞，秋水共长天一色。</View>
</View>
```

## Text Alignment

使用 `textLeft`、`textCenter`、`textRight`、`textJustify` 控制文本的对齐方式。

```jsx live
<View f-20>
  <View textLeft>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View textCenter>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View textRight>落霞与孤鹜齐飞，秋水共长天一色。</View>
  <View textJustify>落霞与孤鹜齐飞，秋水共长天一色。</View>
</View>
```

## Text Color

### 使用设计系统的颜色

```jsx live
<View f-20>
  <Text gray60>王勃</Text>
  <Text colorPrimary> 李白</Text>
  <Text green40> 杜甫</Text>
</View>
```

所有的预设颜色：[Color palette](/docs/design-system-color)

### 使用 `color` props

```jsx live
<View f-20>
  <Text c="#999">王勃</Text>
  <Text c="deeppink"> 李白</Text>
  <Text c="#20B2AA"> 杜甫</Text>
</View>
```

### 速改变颜色透明度

```jsx live
<View row evenly>
  <View c="#ed8936">#ed8936</View>
  <View c="#ed8936.8">#ed8936.8</View>
  <View c="#ed8936.6">#ed8936.6</View>
  <View c="#ed8936.4">#ed8936.4</View>
  <View c="#ed8936.2">#ed8936.4</View>
</View>
```
