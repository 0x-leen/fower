Styli 是一个 CSS-in-JS library，当你在用 Styli 编写样式时，你是在写 JS，而不是在写 CSS。

相比 CSS，JS 更加灵活，图灵完备，能使用变量、条件、函数、作用域等一切编程语言所拥有的东西。

## Styli 和 JSX

JSX 是一个有非常优秀的技术，它可以代替 HTML，让你声明式地描述 UI 结构，并且非常直观优雅。Styli 深度依赖了 JSX，它们结合一起后，可以代替原生的 CSS。使用 Styli 和 JSX，你可以在一个地方声明式地编写 UI 结构和样式。开发过程中，你只需要写 JS 代码，如果你是 JS 老鸟，我相信你会喜欢这种开发模式。

我相信用内聚的方式管理 UI 的结构和样式，会逐渐成为一种潮流。CSS in JS 在蓬勃发展中，也侧面说明有相当多的人希望能用 JS 来编写样式。

## 变量

```jsx
const textColor = '#000'

<Text c={textColor}>Hello word!</Text>
```

## 条件

```jsx
<Text c={isTrue ? 'red' : 'green'}>Hello word!</Text>
```

## 函数

```tsx
<Text c={() => '#000'}>Hello word!</Text>
```

Styli 是一种 CSS in JS 解决方案，当你在构建 UI 时，本质上你是在写 JS，而不是在写 CSS，JS 比 CSS 更灵活，特别实在状态和样式结合的场景，Styli 能更加优雅的处理。
