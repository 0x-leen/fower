id: height
title: 高
---

使用原子 Props 控制元素的高度。高度是高频率使用的属性，所以我们使用缩写。

## 固定高度

使用 `h-{size}` 格式的 Props 可以控制元素的高度。

```jsx live
<View left>
  <View h-50 bgRed20>
    Hello
  </View>
  <View h-5rem bgYellow20>
    Hello
  </View>
  <View h={50} bgGreen20>
    Hello
  </View>
  <View h="100px" bgBlue20>
    Hello
  </View>
  <View h="20%" bgPurple20>
    Hello
  </View>
</View>
```

## 最小高度

使用 `minH-{size}` 格式的 Props 可以控制元素的最小高度。

```jsx live
<View left>
  <View w-50 minH-50 bgRed20>
    Hello
  </View>
</View>
```

## 最大高度

使用 `maxH-{size}` 格式的 Props 可以控制元素的最大高度。

```jsx live
<View left>
  <View w-50 maxH-50 bgRed20>
    Hello
  </View>
</View>
```
