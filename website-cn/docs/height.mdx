## 固定高度

使用 `h-{size}` 格式的 Props 可以控制元素的高度。

:::info
`height` 属性是高频使用的样式，所以简化为 `h`。
:::

```jsx live
<Wrapper>
  <View h-50 bgRed20>
    Hello
  </View>
  <View h-5rem bgYellow20>
    Hello
  </View>
  <View h={50} bgGreen20>
    Hello
  </View>
  <View h="100px" bgBlue20>
    Hello
  </View>
  <View h="20%" bgPurple20>
    Hello
  </View>
</Wrapper>
```

## 最小高度

使用 `minH-{size}` 格式的 Props 可以控制元素的最小高度。

```jsx live
<Wrapper>
  <View w-50 minH-50 bgRed20>
    Hello
  </View>
</Wrapper>
```

## 最大高度

使用 `maxH-{size}` 格式的 Props 可以控制元素的最大高度。

```jsx live
<Wrapper>
  <View w-50 maxH-50 bgRed20>
    Hello
  </View>
</Wrapper>
```

## 伪类

styli 基本支持了所有的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View center s-100 bgBlue30 h--hover={120}>
    hover
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View center w-200 h={[30, 50, 70, 90]} bgBlue30>
    media query
  </View>
</Wrapper>
```
