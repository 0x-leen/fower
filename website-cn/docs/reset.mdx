id: reset
title: reset 重置
---

`reset` 属性可以快捷重置浏览器默认样式。

## 默认规则

使用默认 reset 规则

:::info
默认规则为 [normalize.css](https://github.com/necolas/normalize.css) 的 Styli 实现
:::

```tsx
const Body = styled('body')

<Body reset>
  // other code
</Body>
```

## 自定义规则

为 reset 传入对象值，即可使用自定义 reset 规则

```tsx
const Body = styled('body')

<Body reset={
  {
    '*': {
      margin: 0,
      padding: 0,
    },
  }
}>
  // other code
</Body>
```

**注意: reset 为全局注入属性。组件与 reset 没有对应关系，所以 reset 属性值不能进行 true/false 切换操作。**
