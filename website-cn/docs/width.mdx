使用原子 Props 控制元素的宽度。

宽度是高频率使用的属性，所以我们使用缩写。

## 固定宽度

使用 `w-{size}` 格式的 Props 可以控制元素的宽度。

```jsx live
<View left>
  <View w-50 bgRed20>
    Hello
  </View>
  <View w-5rem bgYellow20>
    Hello
  </View>
  <View w={50} bgGreen20>
    Hello
  </View>
  <View w="100px" bgBlue20>
    Hello
  </View>
  <View w="20%" bgPurple20>
    Hello
  </View>
</View>
```

## 最小宽度

使用 `minW-{size}` 格式的 Props 可以控制元素的最小宽度。

```jsx live
<View minW-100 bgRed20 inlineBlock>
  Hello
</View>
```

## 最大宽度

使用 `maxW-{size}` 格式的 Props 可以控制元素的最大宽度。

```jsx live
<View maxW-100 bgRed20>
  Hello
</View>
```
