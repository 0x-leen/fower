使用 `p{t|r|b|l|x|y}-{size}(-{size})` 格式的 Props 可以控制元素的 padding。

## Padding Props

:::info
`padding` 属性是高频使用的样式，所以简化为 `p`。
:::

- `p` padding
- `pt` padding-top
- `pr` padding-right
- `pb` padding-bottom
- `pl` padding-left
- `px` padding-left and padding-right
- `py` padding-top and padding-bottom

## 设置四周的 padding

使用 `p-{size}` 格式的 Props 可以控制元素四周的 padding。

```jsx live
<Wrapper>
  <Text p-10 bgBlue30>
    Hello
  </Text>
</Wrapper>
```

## 设置一边的 padding

使用 `p{t|r|b|l}-{size}` 格式的 Props 可以控制元素某一边的 padding。

```jsx live
<Wrapper>
  <View debug pt-10>
    Hello
  </View>
  <View debug pr-10>
    Hello
  </View>
  <View debug pb-10>
    Hello
  </View>
  <View debug pl-10>
    Hello
  </View>
</Wrapper>
```

## 设置水平方向的 padding

使用 `px-{size}` 格式的 Props 可以控制元素水平方向的 padding。

```jsx live
<Wrapper>
  <Text px-10 bgBlue30>
    Hello
  </Text>
</Wrapper>
```

## 设置垂直方向的 padding

使用 `py-{size}` 格式的 Props 可以控制元素垂直方向的 padding。

```jsx live
<Wrapper>
  <Text py-10 bgBlue30>
    Hello
  </Text>
</Wrapper>
```

## 设置水平和垂直方向的 padding

使用 `p-{size}-{size}` 格式的 Props 可以控制元素垂直方向的 padding。

```jsx live
<Wrapper>
  <Text p-20-5 bgBlue30>
    Hello
  </Text>
</Wrapper>
```

## 伪类

styli 基本支持了所有的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View center s-100 bgBlue30 p-20 p--hover={10}>
    <View center bgBlue40 s-100p>
      hover
    </View>
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View center h-100 p={[5, 10, 15, 20]} bgBlue30>
    <View center bgBlue40 s-100p>
      media query
    </View>
  </View>
</Wrapper>
```
