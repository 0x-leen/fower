使用 `m{t|r|b|l|x|y}-{size}-{size}` 格式的 Props 可以控制元素的 margin。

## Margin Props

margin 是高频率使用的属性，所以我们使用缩写。

- `m` margin
- `mt` margin-top
- `mr` margin-right
- `mb` margin-bottom
- `ml` margin-left
- `mx` margin-left and margin-right
- `my` margin-top and margin-bottom

## 设置四周的 margin

使用 `m-{size}` 格式的 Props 可以控制元素四周的 margin。

```jsx live
<View bgGray10 left>
  <View bgBlue30 s-100 m-0>
    Hello
  </View>
  <View bgBlue30 s-100 m={20}>
    Hello
  </View>
  <View bgBlue30 s-100 m="40px">
    Hello
  </View>
  <View bgBlue30 s-100 m="10px 20px">
    Hello
  </View>
</View>
```

## 设置一边的 margin

使用 `m{t|r|b|l}-{size}` 格式的 Props 可以控制元素某一边的 margin。

```jsx live
<View bgGray10 left>
  <View ml-10 bgBlue30 s-100>
    Hello
  </View>
  <View mt-10 bgTeal20 s-100>
    Hello
  </View>
  <View mr-10 bgRed20 s-100>
    Hello
  </View>
  <View mb-10 bgYellow20 s-100>
    Hello
  </View>
</View>
```

## 设置水平方向的 margin

使用 `px-{size}` 格式的 Props 可以控制元素水平方向的 margin。

```jsx live
<View bgBlue30 left>
  <View mx-10 s-100 bgBlue50>
    Hello
  </View>
</View>
```

## 设置垂直方向的 margin

使用 `my-{size}` 格式的 Props 可以控制元素垂直方向的 margin。

```jsx live
<View bgBlue30 left>
  <View my-10 s-100 bgBlue50>
    Hello
  </View>
</View>
```

## 设置负 margin

使用 `m{t|r|b|l|x|y}--{size}` 格式的 Props 可以设置元素的 负 margin。

当然，你也可以使用 `ml={-10}` 这样的方式为元素设置负 margin。

```jsx live
<View p-50 debugChildren>
  <View ml--10>Hello</View>
  <View mr--10px>Hello</View>
  <View mt--1rem>Hello</View>
  <View mb={-10}>Hello</View>
  <View ml="-10px">Hello</View>
  <View mr="-1rem">Hello</View>
</View>
```

## 设置水平和垂直方向的 margin

使用 `m-{size}-{size}` 格式的 Props 可以控制元素垂直方向的 padding。

```jsx live
<Wrapper>
  <View s-100 debugChildren>
    <View m-20-5 bgBlue30>
      Hello
    </View>
  </View>
  <View s-100 debugChildren>
    <View m--20--5 bgBlue30>
      Hello
    </View>
  </View>
</Wrapper>
```

## 伪类

styli 支持了大量常用的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View s-100 bgBlue30>
    <View center bgBlue40 m-10 m--hover={5}>
      hover
    </View>
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View s-100 bgBlue30>
    <View center bgBlue40 mt={[20, 15, 10, 5]}>
      media query
    </View>
  </View>
</Wrapper>
```
