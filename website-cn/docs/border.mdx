---
id: border
title: border 边框
---

使用 `border{T|R|B|L|X|Y}` 格式的 Props 可以控制元素的 border。

## 四周边框

```jsx live
<Wrapper>
  <View s-80 bgGray20 border="3px solid deeppink">
    border
  </View>
</Wrapper>
```

## 单一方向

```jsx live
<Wrapper>
  <View s-80 bgGray20 borderT="3px solid deeppink">
    borderT
  </View>
  <View s-80 bgGray20 borderR="3px solid deeppink">
    borderR
  </View>
  <View s-80 bgGray20 borderB="3px solid deeppink">
    borderB
  </View>

  <View s-80 bgGray20 borderL="3px solid deeppink">
    borderL
  </View>
</Wrapper>
```

## 伪类

styli 基本支持了所有的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View s-80 bgGray20 border="3px solid deeppink" border--hover="3px solid yellowGreen">
    hover
  </View>
  <View s-80 bgGray20 border="3px solid deeppink" border--active="3px solid yellowGreen">
    active
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View
    center
    h-100
    w-200
    border={['3px solid blue10', '3px solid red30', '3px solid yellow50', '3px solid green70']}
  >
    media query
  </View>
</Wrapper>
```
