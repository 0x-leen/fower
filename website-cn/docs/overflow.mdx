title: overflow 溢出
---

使用`overflow{X|Y}` 系列属性可以快速元素溢出行为

## 溢出

```jsx live
<Wrapper>
  <View s-100 overflow="scroll" bgBlue30 p-10>
    <View s-120 bgRed30></View>
  </View>
</Wrapper>
```

## 横轴溢出

```jsx live
<Wrapper>
  <View s-100 overflowY="hidden" overflowX="scroll" bgBlue30 p-10>
    <View s-120 bgRed30></View>
  </View>
</Wrapper>
```

## 纵轴溢出

```jsx live
<Wrapper>
  <View s-100 overflowX="hidden" overflowY="scroll" bgBlue30 p-10>
    <View s-120 bgRed30></View>
  </View>
</Wrapper>
```

## 快捷属性

为了提高开发效率，Styli 也提供了大量快捷属性。

```jsx live
<Wrapper>
  <View s-100 oyHidden oxScroll bgBlue30 p-10>
    <View s-120 bgRed30></View>
  </View>
</Wrapper>
```

> - oVisible 是 `overflow: visible`
> - oxVisible 是 `overflowX: visible`
> - oyVisible 是 `overflowY: visible`
> - oHidden 是 `overflow: visible`
> - oxHidden 是 `overflowX: visible`
> - oyHidden 是 `overflowY: visible`
> - oScroll 是 `overflow: scroll`
> - oxScroll 是 `overflowX: scroll`
> - oyScroll 是 `overflowY: scroll`
> - oAuto 是 `overflow: auto`
> - oxAuto 是 `overflowX: auto`
> - oyAuto 是 `overflowY: auto`
