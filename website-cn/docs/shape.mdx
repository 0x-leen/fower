## 正方形

使用 `s-{size}` 格式的 Props 快速设置元素为正方形。

```jsx live
<Wrapper>
  <View s-50 bgYellow20>
    Hello
  </View>
  <View s-5rem bgGreen20>
    Hello
  </View>
  <View s={100} bgBlue20>
    Hello
  </View>
  <View s="150px" bgPurple20>
    Hello
  </View>
</Wrapper>
```

## 长方形

使用 `s-{width}-{height}` 格式的 Props 快速设置元素为长方形。

```jsx live
<Wrapper>
  <View s-50-50 bgYellow20>
    Hello
  </View>
  <View s-50-5rem bgGreen20>
    Hello
  </View>
  <View s-5rem-5rem bgBlue20>
    Hello
  </View>
  <View s-50-150 bgPurple20>
    Hello
  </View>
</Wrapper>
```

## 圆

使用 `circle-{size}` 格式的 Props 快速设置元素为圆。

```jsx live
<Wrapper>
  <View circle-50 bgYellow20 center>
    Hello
  </View>
  <View circle-5rem bgGreen20 center>
    Hello
  </View>
  <View circle={100} bgBlue20 center>
    Hello
  </View>
  <View circle="150px" bgPurple20 center>
    Hello
  </View>
</Wrapper>
```

## 伪类

styli 基本支持了所有的伪类。

> Styli 支持的伪类：[查看](/docs/pseudo)

```jsx live
<Wrapper>
  <View center s-100 bgBlue30 s--hover={120}>
    hover
  </View>
</Wrapper>
```

## 媒体查询

数组格式的 value 能让您便捷的使用媒体查询能力

```jsx live
<Wrapper>
  <View center circle={[30, 50, 70, 90]} bgBlue30>
    media query
  </View>
</Wrapper>
```
