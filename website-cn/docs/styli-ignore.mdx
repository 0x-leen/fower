---
id: styli-ignore
title: styliIgnore 属性忽略
---

Styli 通过各种插件为组件注册属性，提供能力。但插件提供的属性与组件属性冲突时，使用 `styliIgnore` 属性可以让 Styli 忽略对该属性的处理。

示例

```tsx
import { styled } from '@styli/react'

// 自定义组件提供了一个 color prop
const Test = (props) => {
  const { color, className } = props
  return (
    <div className={className}>
      { color }
    </div>
  )
}

// 赋予 Test 组件 Styli 能力
const StyledTest = styled(Test)

// Styli 将对 color 属性进行拦截并处理，最终生成一个 className 注入到 Test 组件中
<StyledTest color="red" />

// Styli 将忽略 color 属性，并将其传递到 Test 组件中
<StyledTest styliIgnore={['color']} color="red" />
```
